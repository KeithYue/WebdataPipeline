　　说起世界杯，必不可少的就是夜宵，还记得4年前的德国世界杯吗？为了看球变成了两只熊猫眼~如今四年一度的精彩世界杯又要上演了，各位球迷们准备好卯足精神熬夜了嘛！啤酒小龙虾已经out了，这些东西不仅容易发胖，还特别不健康，也不能帮助消除疲劳哦！为了让世界杯看的更有滋有味，LZ盘点了几道适合看球时吃的健康小零食，让大家通宵也疯狂！<br/>　　经典蔬菜沙拉<br/>　　营养<a href="http://bbs.hexun.com/search/?q=%D6%B8%CA%FD&type=1&Submit=" target="_blank">指数</a>：4颗星<br/>　　美味又容易上手的小食，懒人必备呢！它不必加热，可以最大限度的保持住蔬菜中的各种营养不至于被破坏或流失。并且3种以上的蔬菜同食，在营养上也会起到很好的互补作用。减肥、美容、养眼的作用非常明显呢~这样熬夜就不怕对皮肤不好啦！<br/>　　<img src="http://i04.c.aliimg.com/img/ibank/2014/565/795/1489597565_1707257441.jpg" onerror="fixImg(this)"/><br/>　　煎茄饼<br/>　　营养指数：3颗星<br/>　　茄饼现煎现吃才最好，脆脆的外皮包裹住软糯的茄子，入口即化，作为看世界杯的小食再合适不过了，一个接一个完全停不下来嘛！茄子非常适合夏天食用，有助于清热解暑。多种维生素、脂肪、蛋白质、糖及矿物质富含其中，是很多蔬菜不能比的。其中的维生素E还可以抗衰老，对于熬夜的童鞋来说非常健康！只不过做法是煎的，所以也不能太贪嘴哦！<br/>　　<img src="http://i00.c.aliimg.com/img/ibank/2014/765/795/1489597567_1707257441.jpg" onerror="fixImg(this)"/><br/>　　奇异果酸奶<br/>　　营养指数：5颗星<br/>　　健康的酸奶搭配水果之王Zespri奇异果，是营养满分的夜宵甜点哦！口味滑爽的酸奶搭配水润香甜的Zespri奇异果，清爽微酸，吃下去沁人心脾。细腻柔滑绵密的多层次口感，让人欲罢不能呢~酸奶有改善皮肤细胞活性，延缓皮肤衰老，增强皮肤张力，刺激皮肤新陈代谢、保持皮肤润泽细腻的作用。而Zespri奇异果含有特别的奇异果酵素，能促进肠道蠕动，让身体毒素排光光。其中的维生素C能有效抑制皮肤内的氧化作用，干扰黑色素的行程，预防色素沉淀，保持皮肤白皙。这样即使通宵也不怕第二天气色不好啦！<br/>　　<img src="http://i02.c.aliimg.com/img/ibank/2014/865/795/1489597568_1707257441.jpg" onerror="fixImg(this)"/><br/>　　牛奶南瓜羹<br/>　　营养指数：4颗星<br/>　　细腻绵滑的牛奶南瓜羹很适合夜晚食用，美味可口且奶香味十足。南瓜是健康的粗粮，纤维多多，吃了会很有饱腹感呢。丰富的钾、钙、镁不仅可以促进胃消化，还能美容抗痘。牛奶的功效就不多说啦，适合各个人群，熬夜必备呢！<br/>　　<img src="http://i04.c.aliimg.com/img/ibank/2014/965/795/1489597569_1707257441.jpg" onerror="fixImg(this)"/><br/>　　Zespri阳光金果<br/>　　营养指数：5颗星<br/>　　水果也是看球必不可少的小食，除了常吃的苹果、香蕉、西瓜等还有一种新上市的Zespri阳光金果非常适合看球时吃呢！小小的个头，拥有花蜜般甘甜的汁液，口感浓郁，淡淡的奶香味和水果清香结合在一起，令人陶醉。球赛往往看到一半会没有精神，这时候吃一颗阳光金果，微酸的口感瞬间能让身体清醒，相当于10个苹果的营养密度，瞬间帮助身体补给熬夜流失的养分，实在不可多得呢~<br/>　　<img src="http://i02.c.aliimg.com/img/ibank/2014/175/795/1489597571_1707257441.jpg" onerror="fixImg(this)"/><br/>　　不知道大家对LZ盘点的这几款小食还满意吗？让它们的营养陪伴你度过世界杯的狂欢之夜吧~
