<div class="txtmain">
　　首先是债务问题。国内有天量的政府债和企业债，违约似乎是迟早的事。可是从08年到现在，政府和企业都活的好好的，为什么？<br/>　　因为大陆政府具有无限的偿债能力，或者说印钞偿债能力。中国社会所产生的一切的债务到最后全部可以被央行收购，然后央行再把这些债务一笔勾销。这就是经济问题政治解决的途径之一。而且央行收购这些债务都不需要全部印成钞票来收购，只需要在账面上办理债权转让手续就行，从而降低通胀压力。所以说债务问题根本难不倒政府。也可以进一步说，流动性问题也根本不是问题，因为政府手里有筹码，只要他们愿意，钱随时都可以印。印多印少而已。这1、20年来，政府一直都是这么解决问题的。<br/>　　这样就带来了第二个问题，通胀问题。钱毕竟是印出来了，而且越印越多，印钞的速度肯定超过了居民收入的实际增长速度。那么物价应该疯涨，涨到超出居民实际购买力才对。可是这些都没有发生，为什么？很简单，我国是一个管制型政府，政府管一切。政府始终管制着各类商品的价格，尤其是<a href="http://bbs.hexun.com/search/?q=%C5%A9%D2%B5&type=1" target="_blank">农产品</a>和基本日用品及其原材料的价格。<br/>　　在基本生活资料的供应方面，政府通过合作社，通过牌照的发放、通过控制流通渠道等等手段，严格限制了价格的上涨——这从不少老百姓还能消费嬡婔迪勊提高夫妻生活质量可见一斑，经济虽然很坏，老百姓的生活却没受到大的影响。因为大部分民众的消费需求都不过是基本生活消费，因为中国的中低收入阶层占了居民人口的绝大多数。客观的讲，价格管制已经比计划经济时代的凭票供应和排队购买要进步了。如果通胀进一步加深或者物价一旦失控，中国又会回到票证时代，粮票、肉票、布票，一切都要凭票供应，买什么都要排队。<br/>　　此外，从另外一方面，政府放开了一部分非生活必需品的价格，从而吸收不断印钞所带来的社会上不断增多的通货。最典型的就是商品房，还有各类奢侈品、豪华轿车等等。所以，从这个意义上说，商品房是不可能总体上全面降价的。什么时候商品房全面降价了，那么全面<a href="http://bbs.hexun.com/search/?bid=&q=%CD%A8%BB%F5%C5%F2%D5%CD&type=2&x=9&y=12" target="_blank">通货膨胀</a>的局面就来到了。商品房如果降到1000元一平方，那么大米也会涨到1000元一斤。<br/>　　这样就带来第三个问题，汇率问题。国内的钱越印越多，对内的物价可以实施管制，那么对外的汇率怎么办？又应该如何实施管控呢？人民币体现在汇率上应该是不断的贬值的，因为人民币经济体的大量坏账、烂帐和反复印钞所产生的水分。但是人民币的汇率几十年来非但没有跌，而且还是涨的，为什么？一方面是以美元为主体的外国货币也同样处在不断地贬值和注水的进程中。只需要看看美元几十年来对<a href="http://bbs.hexun.com/search/?bid=&type=4&q=%BB%C6%BD%F0&x=23&y=5" target="_blank">黄金</a>的比价和美国国债的增长规模，就不难理解这一国际大背景。另外一方面，这一点是最重要的，就是美国一直在逼迫乃至帮助人民币汇率上涨和坚挺。这又要分三点来说明。<br/>　　第一，中国几十年来成为发达国家的加工厂，提供廉价商品和廉价原材料，累积了大量<a href="http://bbs.hexun.com/search/?bid=&type=4&q=%CD%E2%BB%E3&x=19&y=16" target="_blank">外汇</a>，这些外汇支撑了人民币汇率；<br/>　　第二，几十年来外国对大陆的<a href="http://bbs.hexun.com/search/?bid=&type=4&q=%CD%B6%D7%CA&x=26&y=8" target="_blank">投资</a>增长已经成为中国对外经济的主流。外资其实已经开始而且一定将要控制中国的经济。人民币汇率下挫会使得外商投资缩水，这是以美国为首的西方世界坚决不允许的。发达国家依赖中国的廉价商品是次要的，把中国作为资本输出对象才是主要的。有兴趣的人可以计算一下外资几十年来在中国的投资规模，它们的产值和利润，所有中国的显性和隐性外债与之相比都不过时毛毛雨。<br/>　　第三，美国和西方发达国家支持下的人民币互换和人民币国际化，支持了人民币汇率的坚挺。很简单，保人民币汇率就是保国际资本在华投资的资产。收购中国是迟早的事。<br/>　　然后就有第四个问题，那就是谁来养活中国的老百姓，也就是就业和收入增长问题。<br/>　　从政府主导型经济向自由<a href="http://bbs.hexun.com/search/?bid=&type=4&q=%CA%D0%B3%A1&x=15&y=10" target="_blank">市场</a>化经济过度，是中国未来的发展方向。<br/>　　外资控制中国没有错，因为外资不是我们万能的政府，外资是不会管一切的。而且外资是有底线的，那就是市场经济法则。也就是自由交换法则。外资如果想把你卖掉，一定是你自愿的。就像谁都知道银行的信用卡是吸血卡，可信用卡还是没有任何行政命令就普及开来了。<br/>　　国际资本的统治要好过封建专制主义的统治。<br/>　　中国政府收老百姓的房子是靠强拆。外资收老百姓的房子是靠借钱给你，让你用房子抵押，然后还不出钱来收房。如果你赖在房里不走，根据合同，你违约，就拘留。整个过程合理合法，符合程序，文明。
