<div class="txtmain">
　　<strong>现在卧室设计越来越需要创意来打造属于自己的专属空间，接下来我们就来看一下，<br/>　　挑高卧室的设计原则 </strong><br/>　　卧室是人们休息的主要处所，卧室布置得好坏，直接影响到人们的生活、工作和学习，所以卧室也是家庭装修的设计重点之一。挑高卧室设计时要注重实用，其次才是装饰。具体应把握以下原则： <br/>　　1、挑高空间顾名思义就是比一般屋型高的空间，包含我们现在常常看到的3米6、4米2、5米8等屋型，都是属于挑高空间。挑高空间必须视结构体来看屋型<br/>　　是否具有挑高的条件，因为在现今寸土寸金的时代，很多屋主也许并不能接受挑高的型式，挑高同时也代表着部分空间不是以机能性的方式来规划，增加的是更宽阔<br/>　　的视野。如果是独栋透天的屋型，如别墅，挑高的形式又会有不同的方式表现，例如天窗的采光规划等。 <br/>　　2、挑高其实某种形式像有一种抚慰人心的作用，或是一种梦想式的满足，因为挑高本身并不具备实质机能，只是藉由挑高视觉延伸，让心灵有沉淀减压的作用。 <br/>　　3、挑高主要是一种对空间的渴望吧，否则这么多的空间，完全不做实用机能上的设计岂不可惜，然而，挑高空间让住在这里的人，身心获得更大的调节与舒压作用，在某一方面来说，反而更有价值。<br/>　　<strong>挑高卧室的搭配流行趋势  </strong><br/>　　1、挑高其实某种形式像有一种抚慰人心的作用，或是一种梦想式的满足，因为挑高本身并不具备实质机能，只是藉由挑高视觉延伸，让心灵有沉淀减压的作用。目前我正在规划一个长型老屋，屋顶开了三面天窗，阳光宣泄而下，不但给足了采光，光影随时间流动更显趣味。 <br/>　　2、挑高空间位置没有一定要在客厅，只是最好选在靠大面窗的位置，也就是说，靠窗位置采光最好，如果挑高的话，光线就可以充分进入屋内，也不会浪费大面窗的条件。 <br/>　　3、挑高位置大多会决定在客厅，因为客厅是家庭成员活动的重心，其实成员活动大多在餐厅，也可挑高餐厅，当然也可以全部挑高不做夹层，只要屋主有需求，能接受就好了。 <br/>　　4、挑高主要是一种对空间的渴望吧，否则这么多的空间，完全不做实用机能上的设计岂不可惜，然而，挑高空间让住在这里的人，身心获得更大的调节与舒压作用，在某一方面来说，反而更有价值。<br/>　　5、位置不是最重要的，主要还是看屋型条件及需求，我也曾把挑高位置放在收纳空间，只要结构条件上没问题，又能满足动线需求及机能，挑高位置放在哪其实没有一个准则。 <br/>　　6、挑高空间可能针对几个区域，如厨房、卫浴可能需要规划天花板，其它空间都可作为挑高空间，不仅客餐厅、书房都是挑高空间，连卧室也是，营造截然不同的睡眠享受。
