　　从什么时候开始，音乐节再也不是一小撮文艺青年摇滚青年的“私人聚会”，它敞开大门，拥抱各种领域和各种国籍的乐团，转而成为了一项全民运动。不管音乐节还是不是当年的音乐节，五月依旧轰轰烈烈地到来，人群依旧开始沸沸扬扬，只不过，在人数上产生了质的飞跃。<br/>　　<img src="http://i02.c.aliimg.com/img/ibank/2013/233/178/836871332_1829664561.jpg" onerror="fixImg(this)"/><br/>　　你可以看见成群结队的兄弟连、姐妹帮还有一对对拉仇恨的情侣。比起以往T恤、牛仔、长裙的质朴青年，打着小清新重摇滚旗号的时髦人士越来越多了，超短上衣、齐p短裤越来越多了，两臂肌肉、骚包紧身裤也越来越多了。眼球效应，早已经在不知不觉中轰炸全球。<br/>　　无论你是打着个性旗号找妹子约汉子，还是本着一腔未挥发的热血追乐队捧场子，只有一样东西大家都需要——充满力量的Body！前者需要好身材撑场面，后者需要好身体撑体力。趁着还有几天时间，哪怕是临时抱佛脚，咱也来谈谈如何在短时间内获得有效的锻炼成果。<br/>　　<img src="http://i00.c.aliimg.com/img/ibank/2013/529/568/836865925_1829664561.jpg" onerror="fixImg(this)"/><br/>　　宅男宅女变身速成法<br/>　　到了这个时候想要一天练出肌肉和腰身是不可能了，从整体下手，减少肥腻软趴趴的感觉还是比较有希望的。下面就介绍一下整体状态改善的四个步骤：<br/>　　马步增强平衡感：扎马步有助锻炼平衡感，具体做法是，两脚平行站立，与肩同宽，身体挺直，吸气，呼气时膝盖弯曲，将意识集中在肚脐下5厘米的点上，放慢呼吸。<br/>　　后抬腿锻炼臀部：站立时，扶住固定的物体，如桌子、洗碗池等，一条腿缓缓后抬，离地5—10厘米，脚尖绷直，保持这个姿势，深呼吸15—20次，然后轻抬腿20—30次，在抬腿高点处停留2秒，再换腿练习。这个小动作可以锻炼臀部肌肉，保护腰背。<br/>　　收腹减脂法：在桌前、车里或电视机前先深呼吸，放松腹部肌肉，然后呼气，同时尽可能收腹，坚持几秒，感到每块腹肌都在收缩。连续做8次，间隔一会，再重复2次，长期坚持还能预防慢性疾病。<br/>　　<img src="http://i00.c.aliimg.com/img/ibank/2013/629/568/836865926_1829664561.jpg" onerror="fixImg(this)"/><br/>　　型男型女长期雕塑法<br/>　　前面说了想要一天练成魔鬼身材那是痴心妄想，想要成为音乐节上的眼球王还得下点真功夫。本着今年打酱油，明年再战的强烈意志，楼主下面介绍一下健身房或者家庭健身的心得：<br/>　　首先是一套专业教练制定的健身房训练计划，有条件的在家进行也可。<br/>　　周一：胸部训练+30~40分钟有氧+腰腹（椭圆机，慢跑，单车都可以）<br/>　　周二：背部训练+30~40分钟有氧<br/>　　周三：胳膊训练+30~40分钟有氧+腰腹<br/>　　周四：休息<br/>　　周五：腿部训练+20分钟左右有氧+腰腹<br/>　　周六：肩部训练+腰腹<br/>　　其次是健身器械的介绍，一般健身器械的好坏与健身房的高低端程度成正比。总体来说就是国产和国外进口的区别，可别千万小瞧器械间的差别。像美国必确precor之类的高端进口健身器械，特别注重人体工学设计，通常操作起来更加顺畅。比如同样花45分钟锻炼，进口器械能够帮助你保持频率的高效性，这就是因为用起来不吃力，不浪费体能。<br/>　　最后还是提一下健身时的饮食，不管是健身还是日常饮食，楼主都推荐少食多餐。就是说保持一天饮食的总量不变，把三餐再细分，比如分成5~6餐。
