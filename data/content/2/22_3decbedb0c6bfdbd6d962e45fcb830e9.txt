<div class="txtmain">
　　面膜的出现带给了全世界女性崭新的护肤理念，不知不觉中敷面膜已经成为了女性护肤程序中必不可少的一部分，但其实敷面膜也是很有学问的。<br/>　　一、各种肤质面膜选择的原则</p><p>　　1、“膜”法理“油”</p><p>　　大部分油性肌肤，都少有雀斑的烦恼，平日要加强清洁程度，和你可使用深层清洁面膜来改善出油的烦恼。每周至少两次。使用深层清洁面膜。如果“T”字部位易出油，不妨适当地用控油产品，每天将面膜敷在最常出油的鼻子中央“T”字部位，有针对性地护理。</p><p>　　2、 “膜”法抗“旱”</p><p>　　干燥的皮肤还容易出现脱皮情况，太强烈的去角质或深层清洁，会连正常角质除掉，使皮肤抵抗力变弱。应该选用补水并具美白功效的面膜。通常此类面膜都性质温和，最适合干性皮肤，能有效地清除毛孔内的污垢，去除死皮，促进皮肤细胞再生，使肌肤清新柔嫩洁白细滑，更年轻亮丽，并减少肌肤水分流失，软化肌肤角质去除表皮老死细胞，增进肌肤弹性。</p><p>　　3、 “膜”法搅“混”</p><p>　　混合性皮肤通常“T”字部位较油，其它部位为中性或干性。而它首要的任务就是加强鼻子中央“T”字部位的清洁。注意，只是清洁“T”字部位，而不是全脸都清洁，以免脸颊呈干燥状的肌肤因洗擦多而出现脱皮现象。需要注意的是，在深度清洁皮肤时要选择不含碱性成分的的面膜。</p><p>　　使用面膜后，通常脸部会较干，因而要注意脸颊的保湿功夫，不妨用美白精华液涂匀全脸。通常精华液较清爽，再以保湿性高的乳液，涂于两颊部位。</p><p>　　4、 “膜”法“敏”赶</p><p>　　最需要注意的是敏感性皮肤，一般的面膜都不能近身，但这也并非“无药可救”，最好选择天然泥面膜。这些面膜取自天然泥之纤细粒子精华成分，温和细腻，在洗去脸部污垢时减少脸部肌肤负担，彻底清洁毛孔内污垢，使皮肤更具透明感。</p><p>　　敏感肤质的人在使用面膜前还要仔细阅读产品使用说明，留意面膜到底应该涂多厚，敷用的时间以及清除的步骤。尤其眼部是敏感的区域，使用面膜之前先涂上眼部护理霜，同时要避免将面膜涂在眼部周围。<br/>　　正确方法<br/>　　1试敏 <br/>　　在使用一款新面膜之前要先做一次试过敏反应的试验，这一步尤其重要，但却最容易被mm们忽略。把少量面膜敷料涂在手背上，等半个小时后清洗干净，如果手背上没有出现红痒反应，即可放心敷在脸上，尤其是自制的面膜，并应该先做过敏测试。<br/>　　2 清洁面部<br/>　　在敷面膜之前一定要先卸掉脸上的妆容，彻底清洁肌肤，如有必要还要先去除角质，洁净的肌肤更有利于吸收面膜中的养分，并且也能避免污垢和灰尘进入毛孔中。<br/>　　3热敷<br/>　　为了起到更好的美容护肤作用，为了将面膜的护肤功效发挥得更好，就要在清洁面部之后，敷面膜之前用一块热毛巾来敷脸3分钟，并且在面部的各种部位进行按摩3～5分钟，可以充分打开毛孔，益于面膜中养分的吸收。<br/>　　4涂抹顺序<br/>　　敷面膜的顺序要按照颈部、下颌、脸颊、鼻子、嘴唇、额头的顺序，并从下向上敷，注意绕开眼睛周围、眉毛、双唇，建议从最干燥的部位开始涂起，先涂t字区，再涂u字区。<br/>　　5 取下<br/>　　在面膜敷够15分钟后，用手指轻轻按下，感觉不黏手了，就可从面膜的边缘开始，从下向上慢慢地取下面膜。通常面膜干燥后会使肌肤紧缩，容易出现皱纹，所以在面膜没有变干燥前就取下，不要敷得时间过长，更不能敷着面膜过夜。<br/>　　6 冷敷<br/>　　取下之后用干净的温水清洗面部，将面膜残留物清洗干净，之后再用冷毛巾敷面片刻，为的是促进毛孔收缩，之后再涂上润肤品。mm们可不要忘记最后的收敛步骤，有些清洁性和去角质性的面膜，如果没有很好的后期处理，很容易造成皮肤伤害。 <br/>　　更多时尚小常识请关注我们微信：nailiao2012

   <div class="txtmain">　　匈牙利的高<a href="http://bbs.hexun.com/search/?q=%BF%C6%BC%BC&type=1&Submit=" target="_blank">科技</a>产业</p><p>　　（一）<a href="http://bbs.hexun.com/search/?q=%B5%E7%D7%D3&type=1" target="_blank">电子</a>业 　　</p><p>　　从上世纪90年代开始，匈牙利成为世界电子工业主要生产基地之一。电子行业吸引的外国<a href="http://bbs.hexun.com/search/?bid=&type=4&q=%CD%B6%D7%CA&x=26&y=8" target="_blank">投资</a>增长迅速，一直保持着10％的增幅，目前年产值在100亿美元左右，占欧盟总产值的4.5％，匈牙利成为中东欧地区最重要的电子工业国家。 </p><p>　　（二）汽车工业 　　1989年后，匈牙利汽车工业成为获得外国直接投资最多的行业，四分之一的FDI流入该行业。与匈整个<a href="http://bbs.hexun.com/search/?q=%D6%C6%D4%EC&type=1" target="_blank">制造</a>业一样，汽车工业也是外资驱动型，主要由外国公司主控，外国公司从1990年代起就看中匈汽车工业的潜力，向其大量投资。</p><p>　　（三）信息通讯技术 　　匈牙利信息通讯技术发展迅速，2003－2006年间平均增长10％，略高于中东欧地区9.6％的平均增幅，远高于欧盟的2.75％。目前匈占本地区信息通讯<a href="http://bbs.hexun.com/search/?bid=&type=4&q=%CA%D0%B3%A1&x=15&y=10" target="_blank">市场</a>18.7％的份额，信息通讯支出人均<a href="http://bbs.hexun.com/search/?q=%D6%B8%CA%FD&type=1&Submit=" target="_blank">指数</a>和占GDP的比例都属第一位。 </p><p>　　（四）生命科学和生物技术 　　匈牙利生命科学研究历史悠久，在世界上也得到了广泛认同。生物技术做为一个行业，10年前在中东欧地区还是一片空白，但经过匈政府的大力扶持和投入，目前生物技术在匈已经呈现迅猛发展的势头.由于政府的介入以及投资者认识到生物技术的发展潜力，核心生物技术公司大量增加，至2005年有50家公司注册，雇员超过800人。在10个新欧盟成员国中，匈生物技术行业实力最强，她的主要目标是成为欧盟生物技术十强之一。 　</p><p>　　五）再生能源 　　匈牙利具有丰富的生物燃料（Biofuel）、生物质(Biomass)、生物<a href="http://bbs.hexun.com/search/?q=%C8%BC%C6%F8&type=1" target="_blank">燃气</a>(Biogas)、地热(Geothermal)、风能和太阳能资源，具有发展再生能源的潜力。虽然目前再生能源在整体能源结构中只占一小部分，匈政府已经制定了长期规划，在10年内达到欧盟对替代能源比例的要求。</p><p>　　北京联络处：</p><p>　　地址：北京市朝阳北路青年路口北青年汇104号楼222   邮编:100123</p><p>　　电话：＋86（0）10 57275220   直线：＋86（0）10 57275221</p><p>　　传真：+86（0）10 57406338    24小时工作手机：13521144009（袁总）</p><p>　　联系人：孙浩       邮箱：mbt_bj@163.com   QQ:793121392</p><p>　　skype:   owt_bj   </p><p>　　波兰联络处：</p><p>　　地址： Warsaw-Ursus,Bohaterow Warszawy 26/2,02-495 Warsaw. </p><p>　　电话、传真：0048-22-7265149    QQ:1210132364  QQ:793121392</p><p>　　MSN:yuanhui200508@hotmail.com    邮箱：mbt_bj@163.com</div>
