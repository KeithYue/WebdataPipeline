　　<strong><font color="#0000ff">06-05大盘【</font><font color="#0000f0">止跌缘于空头良心发现<a href="http://bbs.hexun.com/search/?bid=&type=4&q=%CA%D0%B3%A1&x=15&y=10" target="_blank">市场</a>虽弱但破2000难</font></strong><font color="#0000ff"><strong>】</strong></font><br/>　　======================== 　<br/>　　欲拾千金踏股海，入水方知际无边 　 　　<br/>　　======================== <br/>　　由于假后首日利好也没有能够刺激市场，反而挫伤了市场的做多信心，这直接反映在了周三的市场当中，早盘开盘后市场呈现了明显的资金外流迹象，空头力量不断扩大，并且早盘根本没有止跌的意味，全天也基本形成了单偏的下跌，我们在午间收评的时候说，不要期望市场午后能够有强势反弹，只能期望空头良心发现，最终的事实证明，午后市场基本依靠空头停止抛售，抄底盘让<a href="http://bbs.hexun.com/search/?bid=&type=4&q=%B9%C9%D6%B8&x=20&y=13" target="_blank">股指</a>略有收窄，最终沪指全天收跌13.47点，量能继续保持在600亿以下水平。</p><p>　　从今天的消息面来看，周三市场同样有利好支撑，特别是对相关的板块有利好支撑，但是市场仍然不顾利好消息，或者说市场对利好反映十分平谈，比如前期已经表现较强的网络安全股、软件<a href="http://bbs.hexun.com/search/?q=%BF%C6%BC%BC&type=1&Submit=" target="_blank">科技</a>股、生物医药股和核电题材，纵有习大大力挺这些个股并没有走出持续的强势,但是这些个股仍然将会是未来行情上涨的主流。</p><p>　　从盘面来看，当下市场具有这样的特点：<br/>　　1、热点板块不能持续；近期题材股频现市场，盘中表现都十分好看，但是这些个股持续性不强，很多都是一日游或者半日游，因此想从中赚钱十分的困难。<br/>　　2、权重板块不轻易表态；前期<a href="http://bbs.hexun.com/search/?bid=&q=%C8%A8%D6%D8%B9%C9&type=2&x=30&y=6" target="_blank">权重股</a>主要是以护盘为主，而近期即不做多也不转头做空，反而显然聪明许多，交替表现，一个在空头一个在多头，阵营分化不明显，也导致股指呈现窄幅波动走势。<br/>　　3、利好频现市场股指反映平谈；近期的利好政策不能说不多，但是股指反映都十分的平谈，有时候连个高开都没有，这一方面说明市场弱，另一方面也说明还在质变当中。<br/>　　4、空头力竭多头在等新股落地；其实近期市场表现为空头砸盘力度不够，而多头又压抑，做多心存顾虑重重。</p><p>　　市场这样的特点表明未来的大盘走势，也将有这样的特点，整体表现弱势震荡格局，个股和题材将轮番表现，机会也主要在盘中热点和题材股当中。如今天很多个股也借机洗盘，当然能够探下去收回来的都是好股，好股这个时候一定要表现为抗跌，或者跌下去能够收回来。股指今天的下影线，也表明市场跌穿2000仍十分困难。</p><p>　　个股点评：<br/>　　002416在反弹路上走着，目标60日线不变；<br/>　　002008受股指托累，跌幅稍大，短线变为谨慎<br/>　　002089走势正常，长长的下影线表现，这只是洗盘<br/>　　002595由于周二的上涨无量支持，今天阴包阳，短线谨慎<br/>　　002288短线有继续调整之势，短线需收复十日线和五日线为上策<br/>　　600593反弹虽有阻力，洗洗更健康</p><p>　　====================================================================<br/>　　***以上为个人观点 据此操作 盈亏自负***转载请注明出处*<br/>
