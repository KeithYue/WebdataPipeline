　　2014年4月25日11点20，我家楼下停车场物业保安通过对讲告知我，派出所有人找我。我与父母三人一起下到地下车库后看到黑压压的制服人群，该群人员由园岭派出所彭新天副所长（警号056678）与邱警官带队。彭副所长要求我把停放在我前夫郑宇平车后的小车挪开，我同意并马上开走到其他停车位。<br/>　　停好车后，我跟警察及郑宇平理论，当初离婚后是郑主动把车位交于我使用，停车位的管理费也是我交的。郑在现场诬陷我偷走他的停车卡，我因生气与郑发生争吵。彭副所长不分青红皂白强令手下带我走，我问他你凭什么带我走，并责问他是不是被人收买了，并要求打电话给我的律师。这时彭副所长恼羞成怒，大叫一声：扭住她，手机抢下。数名警察反扭住我的胳膊，我反抗挣扎中，车钥匙手机均掉地上。彭所长又下令抬起来，于是我被英勇的警察抬胳膊抬腿，四肢不能着地的抬起，在此过程中我甚至面朝下，造成缺氧，我大声呼喊头晕，喘不上气，他们仍置之不理。在我多次呼喊下，被面朝下抬着走了近百米，甚至抬的路途中有磕碰撞伤。彭所长才下话让他们把我放下，我坐在地库的楼梯上，拼命的喘气，还没等我气喘顺。彭所长又大喊抬起来，又抬了一会，楼梯快走完，要到小区公共花园时，彭所长才又下令将我放下。<br/>　　见到女儿被抓，我父母着急想去帮忙，彭副所长又下令，让四名协警把他们反扭按住。二老都是近七十岁，而且父亲高血压脑出血住院三次，母亲也患有糖尿病及心脏病。两位老人双臂一直被扭住从地库押着走上楼梯到了小区花园，这时母亲感觉头晕，身体发抖，心跳过速，跟协警协商：“我头晕发抖，你们松开，我自己走。”协警请示后说领导不同意。一直强拉住拖行穿过小区公共花园走到路边的警车处，路途中有很多邻居，保安及物业管理处人员目睹了我们三人被强行扭送的过程。<br/>　　被抓到派出所后，我们被关在调解室等待，我多次要求打电话给我律师进行法律咨询，均被无理拒绝。同时，也询问凭什么要抓我们，邱警官对我说你太不给我们领导面子，难道不给领导面子是抓我们的理由吗？在等待过程中，我发现全身酸痛，胳膊，眼睛都有不同程度的伤痕，要求拍照留底。中午所有人都去吃饭，只有我们三个一直在傻等，由于事发突然，身上都没带钱，甚至连叫外卖都不行。郑却在楼上吃着盒饭做着笔录（我去上卫生间时路过他做笔录的办公室亲眼所见）。<br/>　　大约一两点钟，终于有民警来给我做笔录。他出示了《中华人民共和国治安管理处罚法》第二十一条和第二十六条，说我涉嫌侵占公私财物，我才正式知道我是被以什么罪名强制传唤，而在此之前没有任何人出示传唤通知或者口头告知我犯了什么法要传唤。我笔录中告知他：我和郑宇平原是夫妻，13年2月26日离婚，在离婚协议中我们分配了夫妻共有财产，当时住宅分给了我，楼下的车位是属于我家购买的私家车位。我当时被郑误导以为该车位是属于该房子的，离婚协议只写了住宅的归属，没有提到楼下的车位。后来他于13年7月8日搬走时，我才知这个车位是单独购买，跟他理论，他同意把车位给我使用，由我交全部的车位管理费用，由于郑生性心眼极小，对钱财过于在意抠门，我赌气主动给他写了借条，大概内容是借他车位，放一百个心，不会霸占他车位。（这张借条在郑做笔录时也提供给了民警，这张借条也完全可以说明他说的我偷的停车卡是诬陷，这是一场真正的民事纠纷。）13年8月份他把卡交给我，由我向物业管理处交费到14年底，物业管理处也开了发票。14年2月份，我因发现郑在离婚前转移隐藏了夫妻共有财产，将其起诉到福田法院。郑恼羞成怒，大闹物业管理处，辱骂保安队长，并诬陷说是管理处把停车卡给我使用，诬陷我经常拿刀来管理处闹事，要求立即停我的卡。物业管理处不同意，说是我交的费用，而且从没见过我，你们的家事纠纷自己解决。在此后郑又多次闹物业，管理处劝说他，你已搬走，车也不停在小区，你既然给你前妻使用，就应该守信，但他不听，找了园岭派出所领导过来施压，物业无耐又给他办了张卡，但郑还要求必须把我的卡取消。这时物业管理处的主任才打电话跟我沟通，但我无法同意，因为这张停车卡是郑亲自给我使用，我交费到年底，他也是知情认可的，不是他想怎么就怎么样，要三方坐下来一起协商，他采取如此激烈手段，我无法接受，但我也跟管理处表态，这个车位我会行使我正常的权利用到我交费的最后一天归还于他，和他的离婚后财产纠纷官司打完，谁的财产就归谁，是他的我也会还给他。管理处同意我继续使用，也不退费，用到费用结束，重新办张普通停车卡。<br/>　　4月15日郑又派他的司机和一林姓男士（该男士是郑在知道我和他打官司后，叫六个黑社会来家闹事，说他欠款84万让我还的其中一位）。他的司机说车位已经给了这位林姓男士，要在我车后面锁车位锁。我不同意，因为小区有规定车位只能卖给本小区业主，而且这个车位是直接跟我家挂接的，就算郑要卖，也要通知我，我有优先购买权，又凭什么在我的车后面安装车位锁，不让我车出去。这两个闹事的人，自己打电话报警，警察来后知道情况后警告了他们，说这是民事纠纷，你们可以去法院起诉，但不能强行锁人家的车。两个人扔下车位锁灰溜溜走了。下午郑又报案，园岭派出所的邱警官先向物业了解了情况，后来家调解，他说郑讲我偷了他的停车卡，我把事情经过讲给他，当时郑的公司职员林某也在场，邱警官也认同这是民事纠纷。4月17日郑把他的车停在了该车位上，并跟我们的女儿说这车他不要了，以后就停这个小区了。由于小区车位紧张，回来晚经常找不到车位，在加上生气，因他经常不让我见孩子。我就把车停在了他的车后面，这么停只影响到他的车一个，并不影响其他用户。当时我把该车的备用钥匙交给保安，也跟物业管理处的主任电话沟通，如果挡了谁的路，就把车钥匙给人家，车磕了碰了都不用负责，如果他想出来，打电话给我，我马上下去把车开走让他出去，物业管理处同意我停在郑的车后面。<br/>　　在15：30我和母亲仍饿着肚子的情况下做完笔录后，派出所还强迫我写下保证书，强迫我母亲写下传唤家属通知书，如果不写就扣留八小时，或者让人来担保。<br/>　　尊敬的市公安局领导，对园岭派出所行为，有以下几个问题希望得到正确回复：<br/>　　一、警察凭什么插手家事纠纷？<br/>　　我与前夫郑宇平关于车位使用权之争，纯属家事引起的民事纠纷。因该车位发生的争执，郑先后闹过管理处、同时报警处理过，之前园岭派出所邱警官也来家调停过，后认为是家庭事务，要求双方通过法院诉讼处理。<br/>　　关于车位的使用，在离婚后一直由我使用，并且郑一直知情且同意我使用，同时我在管理处办理了年卡并缴纳了使用费。因我将郑起诉到了福田法院，在此之后他开始找我麻烦并拿车位使用的问题找事，我也同意使用到我交费的最后一天，今年年底不再使用该车位。对车位的争议，事实如此简单，很单纯的一件民事纠纷，警察凭什么插手家事纠纷？<br/>　　二、警察出警理由能自导自演吗？<br/>　　经我事后调查，因无人报警师出无名，园岭派出所彭副所长打电话给管理处，要求管理处报警。接到派出所电话时，管理处主任拒绝报警说：“她没影响到任何人，车钥匙也交给了保安，郑宇平过来要开走，她也明确表态随时通知她，她都会开走，给他让位。”在这个方法无效的情况下，彭所长又有预谋的，滥用职权，叫来十几个协警，言语上逼我和父母冲动，他们录像拍照，不告知任何理由的情况下强制传唤。派出所自导自演的一曲出警大戏，算不算栽赃陷害？当时这么多人在现场都看到，该停车位上停的明明是郑的车（有照片为证），我的车在他后面，并没停在郑所说的属于他的车位上，他说他要把车开走，我马上把我的车开到其他车位上，给他让路。彭所长仍以涉嫌侵占公私财务强制传唤，欺负我和年近七十岁的父母，造成我们身上多处伤痕，全身酸痛，就算是我真的犯了法，强制传唤我一人即可，为什么要反扭我父母手臂，强制拖行传唤身患疾病的二位老人，并在派出所威胁老人说，你们还不到七十岁，是暴力抗法，我有权拘留你们。<br/>　　三、警察是公仆还是家丁与奴仆？<br/>　　我接管理处通知到地下车库后发现，此次出警规模宏大，拍照的拍照、摄像的摄像、列阵的列阵，初步估计有20余人。后经查看管理处监控，确认此次出警人数不低于20人。不知是什么严重危急的案件，需要出动二十余名警务人员来处理一起民事纠纷，而且部分是临时紧急抽调过来的，我父母在调解室问协警凭什么强制抓我们，协警说别怪我们，我们也不清楚怎么回事，正在外面执勤，领导就把我们都叫过来来。警察插手投诉人的家事纠纷，请问警察是听命于个人还是服务于大众；如此警察为郑宇平一己之私而出动如此之多的警力，请问警察是人民的公仆还是有钱人的家丁与奴仆？彭副所长你是在帮郑泄私愤吗？郑事后跟朋友说就是为了杀杀我的气焰，让我撤诉，你们又为什么这么帮他？<br/>　　四、强制传唤我三人的法律及事实依据<br/>　　我一名女子和两名患病老人怎样在二十余名警务人员面前妨碍公务。强制传唤过程中为什么要对我进行人身伤害，被七八个人强制抬胳膊抬腿，而且是面朝下被抬着走了百余米，这种滋味彭副所长可以亲自尝试下，整个人都是呼吸困难，喘不上气的。根据相关法律规定,传唤期间需提供必要的饮食,为何饭都不给我们吃，我和母亲做完笔录后签字手都是发抖的。在询问调查过程中为何对我三人进行威胁恐吓。<br/>　　五、郑宇平停车堵小区大门数小时未被处理，警察执法公正性何在？<br/>　　今年3月28日下午郑要求管理处收回我的停车年卡并退费给我，管理处拒绝，郑恼羞成怒，将自己的两辆轿车停放在小区车辆出入口长达数小时。该次事件造成交通堵塞、学校放学学生出行不便等严重不良后果，有业主报警后，但园岭派出所未见任何处理。郑开豪车堵小区出入口一事，被当晚<a href="http://bbs.hexun.com/search/?bid=&type=4&q=%C9%EE%DB%DA&x=17&y=13" target="_blank">深圳</a>都市频道第一现场节目组报道，主持人讲他损人不利己，素质太差，违反了治安管理处罚法。这起明显违反治安管理处罚的行为，到目前为止仍未见任何处理。<br/>　　因车位之争，警察不仅不处理扰乱公共秩序的行为，反而偏袒郑插手家事纠纷，执法公正性何在？<br/>　　25日后我的身体多处淤青酸痛，而我的母亲更受了刺激，心脏病发作，一直在家打点滴，至今未愈，左右手腕也被扭伤疼痛，30日还去了医院治疗。作为子女，看到父母被如此羞辱和伤害，深感痛心，此事在整个小区的影响力也很大，很多邻居都不知道实情认为我们犯了什么法，给我们一家造成严重的身体伤害与精神痛苦。对园岭派出所滥用职权充当家奴，不正当介入民事纠纷，执法不公正的行为，我一定要追究到底。恳请贵局领导在查明事实基础上，对充当家奴的警察从严处理，以维护执法为民的正当形象，并给予合理与正确的答复。
