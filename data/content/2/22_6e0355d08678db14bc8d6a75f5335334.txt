<div class="txtmain">
　　我从10年6月开始认真学习股票，还属于次新股民.一个偶然的机会，了解到中国股市的技术核心秘密就是波浪理论。研究学习了两年多了，对大盘预测渐渐已有所心得。这个帖子，会把对大盘的预测和分析方法一点点写出来。</p><p>　　见到此贴的都是有缘的。</p><p>　　毫无疑问，中国股市的技术核心秘密就是波浪理论，但是中国特色的波浪理论，在艾略特波浪理论的基础上有所变化，掌握了这些变化，再加上一段时间的训练，就能较为准确的预测大盘的走势，从而在股市上不断盈利。</p><p>　　经过两年中国特色波浪理论的训练，我已经能够较有把握的在股市盈利，这波中级行情的第一波上攻，我差不多每天满仓进出，到现在的战绩，毛利41%，扣掉9个点跑道费，净利32%。</p><p>　　<img src="http://f200-9.bbs.hexun.com/_i/b/1/d/b1de05ad-be62-4cae-98d8-cce293b91439.jpg" onerror="fixImg(this)"/></p><p>　　<img src="http://f200-9.bbs.hexun.com/_i/1/2/e/12e1e97d-2524-498e-bf53-7c141919cae6.jpg" onerror="fixImg(this)"/></p><p>　　也许马上就有人说单子可以伪造什么的。这是我发的提醒大家买国金证券的帖子。看一下我是什么时候买国金证券的。
</p><p>　　<img src="http://f200-8.bbs.hexun.com/_i/1/1/9/11972e6e-2053-4cba-8b93-f0511f2ce0b0.jpg" onerror="fixImg(this)"/></p><p>　　中国特色波浪理论综述： </p><p>　　先说驱动浪： </p><p>　　按照艾略特波浪理论，驱动浪是有一些必须遵守的规则的， <br/>　　第一，驱动浪的第3浪不能是最短的一浪。 <br/>　　第二，驱动浪的第4浪不能和第1浪顶部重合， <br/>　　第三，第2浪的回调不能低于第1浪的最低点，第4浪的回调不能低于第3浪的最低点 <br/>　　第四，驱动浪的5个子浪必须是同一个等级的。 </p><p>　　经过研究发现，在这里的股市，由5个子浪构成的驱动浪必然会打破以上四个规则之一。比如，第3浪会是最短的浪。为什么？想想就明白了，如果完全遵守波浪理论，就太容易点清波浪了，这是不行的。违背了波浪理论必须遵守的规则，一般水平的人就会被规则所累，永远点不清波浪了。 </p><p>　　这是中国特色波浪理论中的驱动浪的第一个特点. </p><p>　　<br/>　　中国特色驱动浪的第二个特点： </p><p>　　构图原则：驱动浪本质是由5浪构成，如果在图形上很清楚的显出5浪走势，很容易就点清了，这是不行的，所以在构图时，驱动浪从图形上看上去是3浪结构（只是看上去，本质上还是5浪） </p><p>　　记住以上两个特点，就能够点清中国股市的驱动浪了。另外，记住图形，更能帮助你正确点清波浪。</p><p>　　中国特色波浪理论之调整浪： 调整浪的构成比较复杂，划分调整浪时比较容易出错的。 </p><p>　　<br/>　　第一，打破艾略特波浪理论的规则： </p><p>　　调整浪本质是由A，B，C三浪构成， <br/>　　如果A浪的子浪是由3浪构成的，则称为平台型调整浪，按照艾略特波浪理论的原则，平台型调整浪的B浪在接近A浪起点附近结束，而C浪在略微超过A浪终点的位置结束。 <br/>　　如果A浪的子浪是由5浪构成的，则称为锯齿型调整浪，按照艾略特波浪理论的原则，锯齿型调整浪的B浪的高点应明显比A浪的起点低，C浪在远远超过A浪终点的位置结束。 </p><p>　　对于中国股市来说，中国特色的波浪理论，其调整浪的构成必须打破以上规则， </p><p>　　对于平台型调整浪来说， <br/>　　可能B浪的高点明显比A浪的起点低，C浪在远远超过A浪终点的位置结束。 <br/>　　也可能B浪的高点超过A浪的起点，C浪的终点可以超过A浪的终点，也可以不超过， <br/>　　也可能B浪的高点接近A浪的起点，C浪的终点不超过A浪的终点 <br/>　　也可能A浪或者C浪相对整个调整浪图形特别的长。 </p><p>　　对于锯齿型调整浪来说，会有几种情形， <br/>　　第1, B浪的高点接近A浪的起点，C浪的低点略低于A浪的终点。 <br/>　　第2 B浪的高点超过A浪的起点，C浪的终点可以超过A浪的终点，也可以不超过。 <br/>　　第3 C浪的终点不超过A浪的终点 <br/>　　第4， A浪或者C浪的第4子浪特别突出，整个图形看起来像是5浪图形。 <br/>　　第5， A浪或者C浪相对整个调整浪图形来说特别的长。 </p><p>　　<br/>　　第二，中国特色调整浪构图原则：艾略特波浪理论的调整浪本质上是由A，B，C三浪构成的，但是中国特色的调整浪为了让散户点不清浪，调整浪三个子浪中的任意一个子浪会发生变异，使整个调整浪的图形看上去是5个浪组成。 </p><p>　　第1，调整浪的A，B，C三个子浪中的任意一个子浪本身变成复合浪，复合浪可以是abc三浪构成，也可以是1,2,3,4,5五浪构成,其中a浪,b浪,c浪,1浪,2浪,3浪,4浪,5浪和A浪，B浪，C浪的级别完全一样，使整个图形显出完美清晰的5浪结构， </p><p>　　在我两年的学习经历中，调整浪图形显出完美清晰的5浪结构只遇到过几个图形，记住这些图形有好处的。 </p><p>　　第2，调整浪的A，B，C三个子浪中的任意一个子浪本身变异成复合浪，复合浪由a,b,c三浪构成，其中b浪和c浪缩小一个级别。整个调整浪的图形总的看上去还是三浪结构，但图形中总有一个地方看上去有点别扭，这个别扭的地方就是子浪变异的地方。 </p><p>　　第3，调整浪的A，B，C三个子浪中的任意一个子浪本身变异成复合浪，复合浪由a,b,c三浪构成，其中a浪缩小一个级别，b浪和c浪缩小两个级别。或者c浪缩小一个级别，a浪和b浪缩小两个级别。整个调整浪的图形总的看上去还是三浪结构，但图形中总有一个地方看上去有点别扭，这个别扭的地方就是子浪变异的地方。 </p><p>　　<br/>　　第4，调整浪的A浪产生变异，缩小一个级别，通常其对应的C浪长而笔直，或者是调整浪的C浪产生变异，缩小一个级别，其对应的A浪通常长而笔直。 </p><p>　　第5种变异的形态，就是前半段文章里锯齿型调整浪的第4种情形。A浪或者C浪的第4子浪特别突出，使整个调整浪图形看上去像是5浪结构。</p><p>　　<br/>　　有一点必须特别特别注意并记忆：艾略特波浪理论调整浪的C浪必须是由5个子浪构成的驱动浪，但是中国特色的调整浪的C浪可以是驱动浪，也可以是由3个子浪构成的调整浪 </p><p>　　以上为中国特色波浪理论综述，以下慢慢详细图文并茂的解释。如果跟着我的帖子，认真学习波浪理论，用不了两年，也能比较有把握的在股市赚钱的。

   <div class="txtmain">　　实战中技术指标的运用：</p><p>　　kdj指标本质上反应多方力量在多空力量总和中占据的份额。</p><p>　　在实战中kdj最大的作用是用来辅助判断波浪的级别。如图所示，十年级别的红色驱动浪1浪由5个黄色子浪构成，5个黄色子浪都是年线级别的，可以看到每一个年线级别的子浪下面对应的年kdj（特别注意J值）由低位到高位的运动过程。</p><p>　　<img src="http://f200-9.bbs.hexun.com/_i/3/5/e/35e03724-416c-40f1-854d-3a20b26a52a5.jpg" onerror="fixImg(this)"/></p><p>　　<img src="http://f200-9.bbs.hexun.com/_i/f/4/9/f49f0acc-aae8-48e1-a876-0e9e309e73f6.jpg" onerror="fixImg(this)"/></p><p>　　<br/>　　这一波从1949到2444的行情是月线级别的，可以看到月KDJ（特别是J)由低到高的运动过程。</p><p>　　有时候kdj会出现高位或者低位的钝化，这个时候判断波浪的级别需要一些辅助手段。<br/>　　一个是利用时间周期，一般来说一个波浪通常是由5到8根K线组成，比如日线级别的第1浪，（或者第2浪，第3浪等等），在日K线图上由5到8根K线组成，极限的时候也可以是3根或者13根K线。（前提是第1浪本身不包含延长浪）<br/>　　一个是利用幅度，<br/>　　就驱动浪来说，60分钟级别的一浪的幅度一般大于50点，小于100点。<br/>　　日线级别的一浪的幅度一般大于100点，小于200点。<br/>　　周线级别的一浪的幅度一般大于200点，小于400点。<br/>　　月线级别的一浪的幅度一般大于400点，小于800点。<br/>　　就调整浪来说，60分钟级别的一浪的幅度一般大于30点，小于60点。<br/>　　日线级别的一浪的幅度一般大于60点，小于120点。<br/>　　周线级别的一浪的幅度一般大于120点，小于250点。<br/>　　月线级别的一浪的幅度一般大于200点，小于500点。<br/>　　（注意：调整浪的幅度不如驱动浪的幅度那么遵守规则）</p><p>　　实战中kdj的j值可以作为买卖信号，j值见底（接近0），加上股价急跌，就是买入信号，反之j值见顶（接近100），加上股价急拉，就是卖出信号。</p><p>　　<img src="http://f200-9.bbs.hexun.com/_i/5/0/c/50c26a89-15db-477c-8761-e6af28417b3a.jpg" onerror="fixImg(this)"/></p><p>　　比如图上所示，大盘产生一波日线级别连着大阴线的急跌（大于100点小于200点），日线kdj的j值到了0，macd产生底背离，白色方框内K线图构筑小型双底，则可以介入等待一波日线级别的反弹。</p><p>　　kdj构筑双头，头肩顶之类的图形，也和K线图的双头，头肩顶等图形一样，能起到判断头部和底部的作用，但是相对来说只起参考意义。<br/>　　kdj的低位金叉和高位死叉作为买卖信号在实战中作用不大，因为这两个买卖信号对于股价变动的反应过于延迟，基本起不到抄底和逃顶的作用。</p><p>　　<br/>　　macd在实战中的唯一作用就是顶背离或者底背离，一般来说，日线图上出现顶背离或者底背离，意味着一波周线级别的下跌或者上攻，周线图上出现顶背离或者底背离，意味着一波月线级别的下跌或者上攻，以此类推。 </p><p>　　<img src="http://f200-9.bbs.hexun.com/_i/b/0/5/b0588b0f-ad71-4c4f-83e2-221f3e314f6d.jpg" onerror="fixImg(this)"/><br/>　　如图所示，macd产生周线级别的底背离，随后大盘产生一波月线级别的上攻。</p><p>　　<img src="http://f200-8.bbs.hexun.com/_i/b/d/5/bd5dadab-9c83-4a89-9d64-644ebac44b61.jpg" onerror="fixImg(this)"/><br/>　　如图所示，macd产生日线级别的底背离，随后大盘产生一波周线级别的上攻。</p><p>　　macd构筑双头，头肩顶之类的图形，也和K线图的双头，头肩顶等图形一样，能起到判断头部和底部的作用，但是对于股价变动的反应太慢，相对来说只起有限的参考意义。<br/>　　macd的低位金叉和高位死叉作为买卖信号在实战中作用不大，因为这两个买卖信号对于股价变动的反应比kdj还慢，基本起不到抄底和逃顶的作用。</p><p>　　一般来说，精确掌握kdj和macd两个指标在实战中的作用，基本就够用了，其余的指标可以只做参考。

   <div class="txtmain">　　实战中均线的运用</p><p>　　均线最主要起支撑和阻力的作用.</p><p>　　<img src="http://f200-8.bbs.hexun.com/_i/1/c/5/1c53b2df-7592-43e3-bcb7-3c9853d10714.jpg" onerror="fixImg(this)"/><br/>　　比如一波日线级别的反弹，就看5日均线，站上5日均线，就代表超短线走强了。如图黄色方框所示，当根据波浪理论判断至少会有日线级别反弹时，大盘一旦站上5日均线，往往意味着日线级别的反弹上攻即将开始。</p><p>　　<br/>　　中线，比如一波月线级别的中级上升行情中，小调整是日线级别的，以10日均线为目标，如图所示，红色方框A,B处的调整是日线级别的，得到了10日线的支撑。</p><p>　　<img src="http://f200-9.bbs.hexun.com/_i/3/b/8/3b8c1fd7-39f4-40c7-b4fa-d98569666ab7.jpg" onerror="fixImg(this)"/><br/>　　月线级别的中级上升行情中，大调整是周线级别的，以20日均线为目标。红色方框C处的调整是周线级别的第4浪，可以看到调整到了20日均线处。</p><p>　　<br/>　　一般来说，个股在中级（月线以上级别）上攻过程中，回调的最大目标位就是20均线，有时为了配合时间周期数字，调整目标位改为21日均线，21日均线称为生命线。破了这条线，基本就代表月线级别的中级行情结束了。

   <div class="txtmain">　　说一下判断现在行情的思路。</p><p>　　<img src="http://f200-9.bbs.hexun.com/_i/c/7/b/c7ba5c6b-0ff9-4c97-b660-fd4903a298c8.jpg" onerror="fixImg(this)"/></p><p>　　首先，看一下年k线图，看一下图形，很清晰的5浪图形，再看一下下面的年kdj，很清楚的，年kdj随着波浪由低到高运行，所以对应的5个浪是年线级别的，而5个红色子浪组成的第1浪当然大一个级别，是十年级别的.</p><p>　　绿色十年级别的第1浪结束后，运行绿色十年级别的第2浪。十年级别的第2浪是调整浪，应该由年线级别的A，年线级别的B，年线级别的C浪三浪构成，根据以前说明过的中国特色波浪理论之调整浪特点，可以知道A，B，C三浪之中的一浪会产生变异。 </p><p>　　<img src="http://f200-9.bbs.hexun.com/_i/7/b/4/7b491413-bd33-4eb4-9ffe-9e44d76a5969.jpg" onerror="fixImg(this)"/></p><p>　　下一步打开月K线图，可以看到从6124到2008年10月的1664点，是一波5浪驱动浪下跌，怎么判断级别？下面的月KDJ产生了非常非常严重低位钝化，这就有了一种可能，这波下跌可能不只是月线级别的，计算一下<a href="http://bbs.hexun.com/search/?q=%B5%E3%CE%BB&type=1&Submit=" target="_blank">点位</a>，一共下跌了4000多点，而一般来说，一波月线级别的驱动浪点数应该是800点上下。这就进一步证明这波下跌驱动浪可能大于月线级别。</p><p>　　计算一下时间周期，一般一个波浪由5到8根k线组成，而这波下跌有13根月K线，它的子浪又没有延长浪，所以这波下跌级别很可能是大于月线级别的。</p><p>　　再看一下这波下跌的5个黄色子浪，计算一下每个黄色子浪的点数，每一个子浪的点数都远大于周线级别的（周线级别的驱动浪点数是400点上下），很明显，5个黄色子浪的级别大于周线级别，应该是月线级别，则这一波红色下跌应该是年线级别的。应该是年线级别的A浪。</p><p>　　再与十年级别的绿色第1浪中的红色第5子浪比较一下，可以看到两个波浪的点数差不多是一个级别的，而红色第5子浪是年线级别的。可以确定这波下跌是年线级别的。</p><p>　　<img src="http://f200-8.bbs.hexun.com/_i/b/0/6/b06ec4a9-3262-4296-af83-b9090737e19d.jpg" onerror="fixImg(this)"/></p><p>　　绿色十年级别的调整浪2浪其中的子浪，年线级别的红色A浪结束后，应该是子浪，年线级别的红色B浪，看一下黄色的a，b，c三浪图形，这是一个经典的调整浪图形，以前曾经多次出现过，应该非常熟悉。看一下下面的月kdj，可以看到月KDJ随着黄色a，b，c三浪由高到低，由低到高的运动，可以判定，黄色a，b，c三浪都是月线级别的。</p><p>　　其中黄色a浪有延长浪，点数比较大，有1800点，时间周期也比较长，有15根K线（同时按照中国特色波浪理论的规则，变异就产生在黄色a浪，以后进一步详细分解）。但是黄色b浪点数是1000点，和月线级别的点数基本相符，时间周期是7根月K线，很标准，黄色c浪点数是700点，时间周期是5根月K线，是标准的月线级别。<br/>　　月线级别的黄色a，b，c三浪构成了年线级别的红色B浪（这里暂时定名为“B”浪）</p><p>　　<img src="http://f200-9.bbs.hexun.com/_i/4/1/2/41273664-e5f4-445b-8541-2ab21405c2c4.jpg" onerror="fixImg(this)"/></p><p>　　红色年线级别的B浪结束后，应该运行年线级别的红色C浪。<br/>　　这里需要说明一下，在艾略特波浪理论里，调整浪A，B，C的C浪必须是5浪驱动浪，但是在中国股市，C浪不一定是5个浪构成的驱动浪，也可以是3个浪构成的调整浪。（把A，B，C三浪中的B浪看做是X浪，那么C浪就可以不必须是5浪构成的驱动浪）<br/>　　看一下黄色的a，b，c三浪图形，这也是一个经典的调整浪图形，以前曾经多次出现过，其中黄色c浪按照中国特色波浪理论的调整浪特点产生了小变异）。<br/>　　黄色a浪和黄色b浪所对应的的月kDJ指标在高位钝化，但是黄色c浪对应的月KDJ指标很清晰的由高位运行到了低位。</p><p>　　黄色a浪本身也是3浪构成的调整浪，运行了520点，符合月线级别的调整浪点数500点。时间周期是3根月K线，略短一些。<br/>　　黄色b浪本身也是3浪构成的调整浪，运行了410点，符合月线级别的调整浪点数500点。时间周期是4根月K线。<br/>　　黄色c浪是5浪构成的驱动浪（有小变异），运行了900点，符合月线级别的驱动浪点数800点。时间周期是9根月K线。<br/>　　可以判定黄色a，b，c浪都是月线级别的，一起构成了年线级别的红色C浪（这里暂时定名为“C”浪）</p><p>　　<br/>　　在红色年线级别的A浪结束后，红色年线级别的B浪（这里暂时定名为“B”浪），红色年线级别的C浪（这里暂时定名为“C”浪）也结束了，是不是十年级别的第2浪调整结束了？就此启动十年级别的第3浪，牛市开始了？<br/>　　认真观察一下，这个红色的A，B，C的3浪图形看上去非常怪异，以前未曾见过，而且红色C浪很短，下跌速度也不快，不具备杀伤力，不能达到使散户绝望的地步，散户没有被赶走，就开启十年级别的第2浪牛市，不合逻辑。<br/>　　进一步的，按照中国特色波浪理论调整浪的特点，十年级别的调整浪2浪由红色年线级别的A，B，C三个子浪构成，红色A，B，C三个子浪中必有一浪产生变异，这个变异我们没有看到。</p><p>　　<br/>　　如果是新一轮十年级别的第3浪牛市开始了，那么毫无疑问，十年级别第3浪的子浪，年线级别的第1浪应该是5个浪构成的驱动浪，但是如图所示，黄色月线级别的a浪，黄色月线级别的b浪，黄色月线级别的c构成了一个年线级别的调整浪，而不是年线级别的驱动浪。很明显，这个年线级别的调整浪不是十年级别第3浪的子浪，年线级别的第1浪。</p><p>　　<img src="http://f200-8.bbs.hexun.com/_i/f/4/8/f481c819-87c1-497f-ad6a-a4f3b5890f21.jpg" onerror="fixImg(this)"/></p><p>　　<br/>　　绿色十年级别的调整浪第2浪，由年线级别的A浪，年线级别的B浪，年线级别的C浪三个子浪构成，按照中国特色波浪理论调整浪的特点，其中一个子浪必须产生变异。在这里，年线级别的子浪B浪产生了变异，变成了由年线级别的（A）浪，年线级别的（B）浪，年线级别的（C）浪构成的复合B浪.</p><p>　　复合B浪中，年线级别的（C）浪由月线级别的黄色a浪，月线级别的黄色b浪，月线级别的黄色c浪构成，可以看到黄色a，b，c三浪构成的图形不是很规则，有一个最有效的解决方法，是看权重板块的个股。</p><p>　　<br/>　　把<a href="http://bbs.hexun.com/search/?q=%D0%F3%C4%C1%D2%B5&type=1&Submit=" target="_blank">煤炭</a>有色（还可以参考一下<a href="http://bbs.hexun.com/search/?q=%B1%A3%CF%D5&type=1" target="_blank">保险</a>券商）的个股的月K线图全部认真看一遍（看的时候同样运用kdj，时间周期和点数判断级别，用经典图形判断驱动浪还是调整浪），可以看到绝大多数煤炭有色个股在大盘相应的时段是在运行清晰的同样是月线级别的a，b，c三浪。</p><p>　　<br/>　　现在可以看明白了，月线级别的黄色a浪，月线级别的黄色b浪，月线级别的黄色c浪，运行结束了，也就是年线级别的(C)浪结束了，接下来从2444开始的是年线级别的调整。<br/>　　大盘还有的跌，出局观望。

   <div class="txtmain">　　实战运用：怎么运用中国特色波浪理论分析现在大盘的走势：</p><p>　　<img src="http://f200-9.bbs.hexun.com/_i/6/d/b/6db7794a-f293-44ba-b6aa-9768188a9390.jpg" onerror="fixImg(this)"/><br/>　　如图所示，从去年12月4日到今年2月4日完成了一波绿色月线级别的驱动浪5浪上涨行情，前面的文章已经解释过并且划分过波浪了，从2月4日开始的应该是一波年线级别的下跌调整。<br/>　　当前的走势，大盘是完成了绿色月线级别的（A）浪，开始运行绿色月线级别的（B）浪横盘震荡反弹，那么怎么运用波浪理论判定绿色月线级别的（A）浪走完了？</p><p>　　<img src="http://f200-8.bbs.hexun.com/_i/f/8/4/f8494581-4fb8-4be8-938c-7e9c512ecf06.jpg" onerror="fixImg(this)"/></p><p>　　首先看这60分钟K线图，如图所示，从2月4日开始的一波黄色下跌a浪，60分钟kdj由高位运行到中低位，并且跌了37点，时间周期运行了6根60分钟k线，符合60分钟级别的调整浪标准，所以a浪是60分钟级别的。</p><p>　　再看黄色的b浪，它由三个白色子浪（a）浪，（b）浪，（c）浪组成，这是一个经典的调整浪图形，以前见过多次，应该背下来了。进一步判断三个白色子浪（a）浪，（b）浪，（c）浪的级别，可以看到三个白色子浪对应的60分钟kdj都是相应的由高位运行到低位，或者由低位运行到高位，再点一下白色（a）浪，（b）浪，（c）浪各自的点数和时间周期，全都符合60分钟级别的波浪标准，可以判定，这三个子浪都是60分钟级别的，所以由三个白色子浪（a）浪，（b）浪，（c）浪构成的黄色b浪应该是日线级别的。</p><p>　　<br/>　　现在看黄色c浪，它对应的kdj低位严重钝化，点一下黄色c浪的点数和时间周期，135点和15根60分钟K线，远远超过了60分钟级别的驱动浪的标准，如果换回到大盘的日K线图，可以看到黄色c浪对应的日线kdj由高位运行到低位，同时c浪的时间周期和点数符合日线级别的标准，所以黄色色c浪应该是日线级别的。</p><p>　　现在总的看一下，由黄色a浪，b浪，c浪构成的图形，这是一个经典的调整浪图形，应该有记忆的。同时根据中国特色波浪理论调整浪的特点，中国特色调整浪的三个子浪中的一个必须产生变异，可以看到黄色a浪产生了变异，a浪的级别本应该和b浪c浪一样，也是日线级别的，但现在变成了60分钟级别的。<br/>　　现在可以得出结论，由变异的a浪，正常的b浪和c浪构成了一个红色周线级别的调整浪A浪。</p><p>　　第二步：</p><p>　　<img src="http://f200-9.bbs.hexun.com/_i/0/4/3/043ea0af-d7c0-44b2-952b-3f88e000491b.jpg" onerror="fixImg(this)"/></p><p>　　<br/>　　下面换到大盘的日k线图，如图所示，我们暂且先别去管绿色方框内的K线图形，看一下红色B浪，可以看到红色B由三个黄色子浪a浪，b浪，c浪组成，这同样是一个经典的调整浪图形。进一步判断三个黄色子浪a浪，b浪，c浪的级别，可以看到三个黄色色子浪对应的日线kdj都是相应的由高位运行到低位，或者由低位运行到高位，再点一下黄色a浪，b浪，c浪各自的点数和时间周期，分别是93点，120点，112点，4根，9根，5根日K线，全都符合日线级别的波浪标准，可以判定，这三个子浪都是日线级别的，所以红色B浪相应的应该是周线级别的。</p><p>　　第三步，看一下红色C浪，它是由三个黄色子浪a浪b浪c浪构成，可以看到这也是个经典调整浪图形，判断一下三个黄色子浪的级别，同样的方法，可以判断黄色a浪和黄色b浪都是日线级别的，而黄色c浪是60分钟级别的。再回忆一下中国特色波浪理论调整浪的特点，中国特色调整浪的三个子浪中的一个必须产生变异，在这里黄色c浪本应和a浪和b浪一样是日线级别的，现在变成了60分钟级别的，由正常的黄色a浪，b浪和变异的黄色c浪构成了一个红色周线级别的调整浪C浪。</p><p>　　第四步：<br/>　　现在综合起来看一下，可以发现由红色周线级别的A浪，红色周线级别的B浪和红色周线级别的C浪组成了绿色月线级别的（A）浪。<br/>　　别忘了，中国特色波浪理论调整浪的特点，中国特色调整浪的三个子浪中的一个必须产生变异。在这里红色周线级别的A浪产生了变异，可以看到绿色方框内大盘多运行了两个日线级别的红色子浪（B）浪和（C）浪，它们和红色A浪组合成一个复合浪。

   <div class="txtmain">　　在我这个贴子里，也许可以学到股市的基本技术分析方法，以后我还会逐步把我对基本面分析方法的理解写出来，但是你在看这些入门知识之前，一定要牢记一点：</p><p>　　无论你的技术分析和基本面分析水平到达了何种境界，它们永远是第二位的。<br/>　　排在第一位的永远是猜到对手下一步会干什么。</p><p>　　资本<a href="http://bbs.hexun.com/search/?bid=&type=4&q=%CA%D0%B3%A1&x=15&y=10" target="_blank">市场</a>，股市的涨跌，归根到底是由人的行为控制的，技术面和基本面，只不过给了资本市场和股市一个涨和跌的理由，仅此而已。一件事情有了发生的理由，但未必就一定会发生，一件事看上去没有发生的理由，但是不代表着它一定不会发生。</p><p>　　所以一定要牢记这一点：在资本市场，排在第一位的永远是猜到对手下一步会干什么。如果你忘记了这一点，无论赚了多少钱，最后的结局必定是倾家荡产。</p><p>　　不要做战国时的赵括，有优秀的天赋，所有的理论知识都彻底理解，但是因为太年轻，不明白人和人之间的竞争中最重要的事永远是猜到对手下一步会干什么。</div>

   <div class="txtmain">　　今天大盘下跌了19点，成交量是844亿。</p><p>　　看一下行业板块资金流向，今天银行，<a href="http://bbs.hexun.com/search/?q=%B1%A3%CF%D5&type=1" target="_blank">保险</a>，有色，<a href="http://bbs.hexun.com/search/?q=%D0%F3%C4%C1%D2%B5&type=1&Submit=" target="_blank">煤炭</a>，<a href="http://bbs.hexun.com/search/?bid=&q=%B7%BF%B5%D8%B2%FA&type=1&x=23&y=11" target="_blank">房地产</a>等权重板块一共流出了25亿，只有证券一个权重板块流进了1.8亿。可以看到在最近大盘的震荡过程中<a href="http://bbs.hexun.com/search/?bid=&type=4&q=%D6%F7%C1%A6&x=24&y=9" target="_blank">主力</a>资金依旧不断流出。</p><p>　　不厌其烦的再划分一下波浪，如图所示，现在大盘运行的是绿色月线级别的(B)浪反弹，今天的下跌是在继续红色周线级别的B浪调整，从分时K线看，明天大盘有继续收小小阴线的可能。</p><p>　　<img src="http://f200-8.bbs.hexun.com/_i/d/e/f/def8f036-ad7a-4e4c-bbb7-648bf071a242.jpg" onerror="fixImg(this)"/></p><p>　　已经解释过了，昨天的阳线反抽，应该就是期指主力在玩杀完空头杀多头的游戏，从盘面走势看，不能排除在红色周线级别调整的尾声，期指主力反过来玩一次杀完多头杀空头的游戏，一根大阴线过后运行红色周线级别C浪急升。所以说超短线和短线操作风险很大的，波段操作才是稳定获利的保证。</p><p>　　<br/>　　盘面虽然有些平淡，不妨可以趁着空闲的时间看些有用的书的，前一次说过了，计算大盘和个股每年的运行区间可以利用现金流量贴现模式的，具体的内容可以看博迪写的两本书，两本书都很有名，可以说是经典教材，网上很容易都能找到。</p><p>　　先看比较容易的《金融学》，书里的知识基本够用了，看懂以后在网上找到申银万国的上证<a href="http://bbs.hexun.com/search/?q=%D6%B8%CA%FD&type=1&Submit=" target="_blank">指数</a>因子分解模型，这个模型就是《金融学》知识的实际运用。掌握这个模型就能计算大盘每年的运行区间了。</p><p>　　想进一步提高就看《<a href="http://bbs.hexun.com/search/?bid=&type=4&q=%CD%B6%D7%CA&x=26&y=8" target="_blank">投资</a>学》，如果英语不错就看原版，中文版的翻译有待改进，《投资学》有两个翻译版本，机械工业出版社的似乎好些，不妨两本对照着看。</div>
