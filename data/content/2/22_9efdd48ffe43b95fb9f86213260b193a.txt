　　夏天想要做一个美白水嫩的“白富美“，肿么少得了VC水果呢？”天然美容“在近几年越来越流行，就连各大顶级护肤品、化妆品都大量地加入新鲜水果的提取物。那这些水果的美白功效到底有多强呢，如何针对自己的肌肤问题补充这些营养丰富的水果呢？快来看看这篇水果VC终极攻略吧~<br/>　　1. 熊猫眼的克星——苹果<br/>　　苹果营养丰富，是一种广泛使用的天然美容品。苹果中含有0.3%的蛋白质，0.4%的脂肪，0.9%的粗纤维和各种矿物质、芳香醇类等。其所含的大量水分和各种保湿因子对皮肤有保湿作用，维他命C能抑制皮肤中黑色素的沉着，是熊猫眼的天然克星哟。将苹果切片后敷在黑眼圈部位也是近年来流行的去黑眼圈妙方<br/>　　<img src="http://f200-8.bbs.hexun.com/_i/5/6/5/565cb309-a2d9-4058-8c52-9de2060fe9e5.jpg" onerror="fixImg(this)"/></p><p>　　<br/>　　2.弹！弹！弹！弹走小细纹——桔子<br/>　　桔皮中的维他命C含量特别高，因此在很多地区，人们用桔皮泡水喝以摄取大量的维C。另外，桔子中所含的有机酸可以增强肌肤弹性哟。每天看到自己脸上的小细纹，是不是恨不得用弹弓弹走它！快用桔皮擦脸可以平抚面部小细纹喔<br/>　　<img src="http://f200-8.bbs.hexun.com/_i/8/c/0/8c03acff-a40b-4aa2-a229-04c446db40cb.jpg" onerror="fixImg(this)"/></p><p>　　<br/>　　3.瘦身圣品——西红柿<br/>　　夏天挺着个小肚腩，穿任何性感的包臀裙和连衣裙都像孕妇！想甩掉身上的小肉肉，当然要吃瘦身圣品西红柿啦~许多模特都靠食用西红柿来保持身材。<br/>　　西红柿营养丰富且热量低，是最佳的减肥餐。它丰富的酸性汁液可以帮你平衡皮肤的PH值。<br/>　　<img src="http://f200-9.bbs.hexun.com/_i/4/3/9/43909e54-5e8f-4e1e-ba7c-f60a360d8617.jpg" onerror="fixImg(this)"/></p><p>　　<br/>　　4.美白神器——蔓越莓<br/>　　夏天紫外线强，肌肤很容易被晒黑，黑色素沉积。蔓越莓富含20倍的VC是黑色素的克星哟~蔓越莓生长在寒冷的北美湿地，全球产区不到4万亩，是很稀有的水果，一般国内鲜少见到。但系！LZ最近发现了TA——康师傅每日C的蔓越莓汁，这款让女神王珞丹与“自黑“说”白白“的美白神器呀！这个蔓越莓汁不仅酸甜爽口，它的形象化身——蔓越莓君也是非常呆萌可爱的哟~广受萌妹纸的喜爱！<br/>　　<img src="http://f200-8.bbs.hexun.com/_i/3/2/d/32dde676-15eb-4cc1-92ab-6c9efa05cfa0.jpg" onerror="fixImg(this)"/></p><p>　　<br/>　　看蔓越莓君和女神王珞丹亲密合照，正能量爆棚有木有！遇上宇宙无敌<a href="http://bbs.hexun.com/search/?bid=&type=4&q=%B3%AC%BC%B6&x=32&y=10" target="_blank">超级</a>萌的蔓越莓果树人，生活怎能不新鲜呢~
