					<h1>评李克强总理考察内蒙时的讲话:释放积极政策信号?</h1>
					<div class="info">2014-05-29 11:45   				<span>来源：中国银河证券股份有限公司</span>
                					<span class="FZ"><span class="fl fz">字号：</span><span class="setFZ setFZ12" rel="txt_bd">12</span> <span class="setFZ setFZ14" rel="txt_bd">14</span></span>

                                                                                                 			<div class="txt_bd lh24 fz14" id="ctrlfscont">
				&nbsp;&nbsp;&nbsp;&nbsp;核心观点:</p><p></p><p>1.事件</p><p></p><p>5月22日至23日,中共中央政治局常委、国务院总理李克强在赤峰市考察调研时说,当前我国经济运行总体平稳,结构出现积极变化,但经济下行压力仍然较大,不能掉以轻心。他指出,金融是经济发展的血液和重要支撑。要针对企业反映的实体经济资金总体紧张特别是小微企业融资难、融资贵等问题,运用适当的政策工具,适时适度预调微调,盘活资金存量,优化金融结构,保持货币信贷合理增长,推进金融改革,营造良好的金融环境</p><p></p><p>2.我们的分析与判断</p><p></p><p>对于总理的讲话,市场的解读是:总理讲话定调货币政策转向宽松。我们的观点如下:</p><p></p><p>(一)经济下行压力。</p><p></p><p>总理的讲话中谈到“当前我国经济运行总体平稳,结构出现积极变化,但经济下行压力仍然较大,不能掉以轻心。”那么,总理担心的经济下行压力指的是什么呢?我们认为有以下三点:第一,投资增速下滑;第二,消费增速下滑;第三,外贸增速下滑。</p><p></p><p>2014年是中国经济旧运行逻辑破裂、新模式将立未立的转折性年代,经济增速的下滑将是大概率事件。因此,我们的判断是:中国经济的最大风险应该是建筑业。建筑业产能过剩严重,负增长将出现失业危机,房地产将进入中长期衰退通道。</p><p></p><p>(二)运用适当的政策工具,适时适度预调微调。</p><p></p><p>当前的金融环境是货币供应增速小幅反弹,货币市场流动性较为充裕。但是,信贷规模扩充乏力,银行对中小企业与小微贷款客户信贷投放量不足。</p><p></p><p>我们认为,从李克强总理的表述中并不能确定“货币政策转向宽松”。但是,仍然释放出积极的政策信号,不排除降准的可能性。</p><p></p><p>3.投资建议</p><p></p><p>在基本面疲弱的情况下,如果政策调整没有加力,则未来市场将继续弱势运行,投资策略上继续以防御为主,机会只是结构性的:一是低估值的金融、家电;二是有催化剂的热点个股,如高铁、军工、油气改革等;三是从调整充分的成长股中寻找优质的股票,如TMT、环保、医药等。			</div>

function copyToClipboard(txt) {  //复制网址
	if(window.clipboardData) {
		window.clipboardData.clearData();
		window.clipboardData.setData("Text", txt);
		alert("复制链接成功！");
	} else if(navigator.userAgent.indexOf("Opera") != -1) {
		window.location = txt;
	} else if (window.netscape) {
		try {
			netscape.security.PrivilegeManager.enablePrivilege("UniversalXPConnect");

			alert(" 被浏览器拒绝！\n请在浏览器地址栏输入'about:config'并回车\n然后将 'signed.applets.codebase_principal_support'设置为'true'");
		}
		var clip = Components.classes['@mozilla.org/widget/clipboard;1'].createInstance(Components.interfaces.nsIClipboard);
		if (!clip)
		return;
		var trans = Components.classes['@mozilla.org/widget/transferable;1'].createInstance(Components.interfaces.nsITransferable);
		if (!trans)
		return;
		trans.addDataFlavor('text/unicode');
		var str = new Object();
		var len = new Object();
		var str = Components.classes["@mozilla.org/supports-string;1"].createInstance(Components.interfaces.nsISupportsString);
		var copytext = txt;
		str.data = copytext;
		trans.setTransferData("text/unicode",str,copytext.length*2);
		var clipid = Components.interfaces.nsIClipboard;
		if (!clip)
		return false;
		clip.setData(trans,null,clipid.kGlobalClipboard);
		alert("复制链接成功！");
	}
