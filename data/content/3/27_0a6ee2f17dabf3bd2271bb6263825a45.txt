					<h1>空头持仓不减反增 空军现四年以来最大集结</h1>
					<div class="info">2014-06-06 14:38   				<span>来源：腾讯财经</span>
                					<span class="FZ"><span class="fl fz">字号：</span><span class="setFZ setFZ12" rel="txt_bd">12</span> <span class="setFZ setFZ14" rel="txt_bd">14</span></span>

                                                                                                 			<div class="txt_bd lh24 fz14" id="ctrlfscont">
				&nbsp;&nbsp;&nbsp;&nbsp;昨日期指总持仓有所下降，但“空军司令”中信期货始终没有停下加仓空单的脚步，空头持仓不减反增。在总持仓持续突破并盘踞历史高位的背景下，空头势力犹如一把悬空利剑，冰冷地压抑在驼背的股指上方。</p><p>数据显示，期指总持仓不断创历史新高，周三收盘170038手，首破17万手。自5月6日以来，在短短1月内，总持仓先后攻破14、15、16和17万四大整数关口，这在期指上市4年多的历史中是罕见的。并且，期间总持仓加仓幅度24%，主力净空单增加幅度却高达65%，空头加仓成为总持仓连创新高的主要推力。目前主力净空单高达23000余手，为历史最高水平，这意味着整体持仓结构偏向空方，空军实现四年多以来的最大集结！</p><p>先不管屡次上演的“持仓魔咒”是否会再度引发大盘波段下跌，单是与沪指量能如此极端背离的数据，就足以让多头心存忌惮。更何况，正如我们所指出的那样，IPO重启近在眼前、IPO预披露马不停蹄，市场供求失衡、投资者信心疲弱。且险资等主力也临阵撤退，除了监管层护盘，多头实在难有底气发动规模行情。而结构性游击战法，或是多头主力避重就轻的不二选择。</p><p>因此，操作上投资者宜放弃主升主跌预期，而以弱平衡的震荡行情思维看待。在结构性分化的震荡中，单纯依据指数操作，如遇急涨则追、急跌则杀，往往易造成“两边挨耳光”的反节奏被动局面；与追涨杀跌相反的另一大忌则是盲目捂股，虽然指数从长期来看已处低位，但并不意味着个股也同样处低位。在08年指数大跌中，如中石油、<SPAN id=stock_sh601919><A class=keyword href="http://cy.stcn.com/S/SH601919" target=_blank>中国远洋</A></SPAN><SPAN id=quote_sh601919></SPAN><SCRIPT language=JavaScript>show_quote("601919","sh","中国远洋","stock");</SCRIPT>、<SPAN id=stock_sh601168><A class=keyword href="http://cy.stcn.com/S/SH601168" target=_blank>西部矿业</A></SPAN><SPAN id=quote_sh601168></SPAN><SCRIPT language=JavaScript>show_quote("601168","sh","西部矿业","stock");</SCRIPT>等仍有腰斩空间，何况那些还高高在上头部隐现的爆炒题材股！</p><p>值得庆幸的是，指数波动性及量能日益低迷，但存量博弈中局部机会仍可挖掘。好股互动实盘中，各支轻基金紧扣“高层级事件驱动+产业资本强庄运作”模式，抓住军改、油改、环保、资本运作等方向，尤其是这些题材要素相互叠加共振的低价盘小品种，进而取得稳定的逆市收益，值得投资者借鉴。			</div>

function copyToClipboard(txt) {  //复制网址
	if(window.clipboardData) {
		window.clipboardData.clearData();
		window.clipboardData.setData("Text", txt);
		alert("复制链接成功！");
	} else if(navigator.userAgent.indexOf("Opera") != -1) {
		window.location = txt;
	} else if (window.netscape) {
		try {
			netscape.security.PrivilegeManager.enablePrivilege("UniversalXPConnect");
		} catch (e) {
			alert(" 被浏览器拒绝！\n请在浏览器地址栏输入'about:config'并回车\n然后将 'signed.applets.codebase_principal_support'设置为'true'");
		}
		var clip = Components.classes['@mozilla.org/widget/clipboard;1'].createInstance(Components.interfaces.nsIClipboard);
		if (!clip)
		return;
		var trans = Components.classes['@mozilla.org/widget/transferable;1'].createInstance(Components.interfaces.nsITransferable);
		if (!trans)
		return;
		trans.addDataFlavor('text/unicode');
		var str = new Object();
		var len = new Object();
		var str = Components.classes["@mozilla.org/supports-string;1"].createInstance(Components.interfaces.nsISupportsString);
		var copytext = txt;
		str.data = copytext;
		trans.setTransferData("text/unicode",str,copytext.length*2);
		var clipid = Components.interfaces.nsIClipboard;
		if (!clip)
		return false;
		clip.setData(trans,null,clipid.kGlobalClipboard);
		alert("复制链接成功！");
	}
