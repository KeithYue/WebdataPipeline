					<h1>“并购狂”谷歌欲颠覆传统电力行业</h1>
					<div class="info">2014-06-13 01:45   				<span>来源：证券时报网</span>
                					<span class="FZ"><span class="fl fz">字号：</span><span class="setFZ setFZ12" rel="txt_bd">12</span> <span class="setFZ setFZ14" rel="txt_bd">14</span></span>
					</div>
				</div>
                                                                                                 			<div class="txt_bd lh24 fz14" id="ctrlfscont">
				<P>证券时报记者 吴家明</P><P>热衷于并购的谷歌，正在把发展方向瞄准多元化，不断开拓新的发展领域。据海外媒体报道，美国谷歌公司正在开发高度智能的供电软件和硬件产品，有望和其他创新公司一道给传统的电力行业带来颠覆性的冲击。</P><P>据悉，美国电力市场规模高达3600多亿美元，谷歌已经决定在这一市场精耕细作，并致力于新一代“智能电网”的开发。谷歌正在开发用于管理电力线路网络和电力基础设施的软件和硬件工具，从而减少浪费和提高运营效率，这一项目由谷歌的能源供应项目团队负责。与此同时，谷歌最近的一则招聘启事也透露，谷歌正在研究提供清洁、低成本能源的创新解决方案，开发相应的技术和产品，通过全新的基础设施满足全球电力需求。</P><P>除了谷歌以外，刚刚在美国上市的家庭能源数据分析公司Opower也在从事“智能电网”业务。不过，前美国能源部部长朱棣文曾对谷歌的“智能电网”业务进行了一番评价：谷歌意识到这里有一个巨大的市场空间，现有的电力公司并未涉足，谷歌已经看到了商机。</P><P>此外，随着太阳能和风力发电越来越普及，谷歌早已对全球范围内的太阳能和风力发电项目投资了10多亿美元。有分析人士表示，其实谷歌、苹果、微软等硅谷巨头都是能源耗费大户，但谷歌在可再生能源领域的表现要比其他竞争对手更有进取心。</P><P>除了比较“高大上”的电力市场，谷歌也在大力开拓企业应用市场。谷歌日前推出了一套一站式商户信息管理工具，这项名为“Google My Business”的服务能帮助中小企业业主通过一个统一入口更新商户信息、添加照片、查看评论以及使用谷歌旗下社交网络Google+。</P><P>谷歌不仅追求多元化，从诞生之初它就像一个“并购狂”一样，血液里似乎天生就涌动着并购的因子。特别是在近期，每隔几天就会传来谷歌并购的消息。就在近日，谷歌公司宣布将以5亿美元的价格收购小型卫星公司Skybox，这也是谷歌今年在航天领域的第二笔重量级收购。今年4月，谷歌已经收购了位于美国新墨西哥州的无人机公司Titan Aerospace。</P><P>智能家居公司Nest Labs、以色列声波身份识别技术初创公司SlickLogin、智能手机管理软件开发商Divide以及智能穿戴企业Jawbone……最近有关谷歌出手并购的对象频频出现。在这背后，“弹药”充足成为谷歌大肆并购的有力保障。谷歌近日公布了有关海外现金用途的更多细节，该公司可能利用海外积累的300亿美元现金展开并购。此外，谷歌在过去短短十几年时间已经花费了约270亿美元完成了上百次的并购。</P>			</div>

function copyToClipboard(txt) {  //复制网址
	if(window.clipboardData) {
		window.clipboardData.clearData();
		window.clipboardData.setData("Text", txt);
		alert("复制链接成功！");
	} else if(navigator.userAgent.indexOf("Opera") != -1) {
		window.location = txt;
	} else if (window.netscape) {
		try {
			netscape.security.PrivilegeManager.enablePrivilege("UniversalXPConnect");
		} catch (e) {
			alert(" 被浏览器拒绝！\n请在浏览器地址栏输入'about:config'并回车\n然后将 'signed.applets.codebase_principal_support'设置为'true'");
		}
		var clip = Components.classes['@mozilla.org/widget/clipboard;1'].createInstance(Components.interfaces.nsIClipboard);
		if (!clip)
		return;
		var trans = Components.classes['@mozilla.org/widget/transferable;1'].createInstance(Components.interfaces.nsITransferable);
		if (!trans)
		return;
		trans.addDataFlavor('text/unicode');
		var str = new Object();
		var len = new Object();
		var str = Components.classes["@mozilla.org/supports-string;1"].createInstance(Components.interfaces.nsISupportsString);
		var copytext = txt;
		str.data = copytext;
		trans.setTransferData("text/unicode",str,copytext.length*2);
		var clipid = Components.interfaces.nsIClipboard;
		if (!clip)
		return false;
		clip.setData(trans,null,clipid.kGlobalClipboard);
		alert("复制链接成功！");
	}
