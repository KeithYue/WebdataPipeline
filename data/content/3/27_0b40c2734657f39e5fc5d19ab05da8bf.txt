					<h1>水泥价格周报:价格继续回落,幅度0.52%</h1>
					<div class="info">2014-06-10 10:11   				<span>来源：中银国际证券有限责任公司</span>
                					<span class="FZ"><span class="fl fz">字号：</span><span class="setFZ setFZ12" rel="txt_bd">12</span> <span class="setFZ setFZ14" rel="txt_bd">14</span></span>

                                                                                                 			<div class="txt_bd lh24 fz14" id="ctrlfscont">
				&nbsp;&nbsp;&nbsp;&nbsp;全国水泥市场价格环比下降0.52%。</p><p></p><p>据数字水泥网的消息显示,上周全国水泥市场价格环比前一周继续回落,幅度为0.52%。价格下跌区域主要有江苏、安徽、江西和广东等部分地区,不同地区价格下调20-30元/吨;价格上涨地区主要是贵州贵阳和黔南地区,上调20元/吨。6月初,国内水泥市场需求延续疲弱态势,加上受高考、农忙和天气等不利因素影响,部分地区水泥价格加速下滑。</p><p></p><p>华北地区价格小幅波动。天津地区稳中略有小幅回落,幅度10元/吨,主要是受下游资金紧张影响,部分企业为增加销量和资金回笼,价格小幅优惠为主。北京地区价格平稳,市场成交情况较前两周好转,但下游资金紧张,水泥企业货款回收压力偏大,另外受砂石骨料供应不足影响,部分搅拌站开工率仍未正常。河北石家庄、保定、唐山等地区受高温、雨水天气的影响,出货情况稍有不足,水泥价格变化不大。</p><p></p><p>东北地区价格平稳运行。东北地区下游需求不温不火,市场成交疲软运行。</p><p></p><p>佳木斯有重点工程项目支撑,市场价格趋强运行,水泥价格继续小幅走高,五月上旬以来累计涨幅30元/吨。沈阳、大连、哈尔滨等地区价格保持平稳。</p><p></p><p>华东地区P.O42.5水泥价格环比前一周下跌3.2%。6月初,江苏南部苏锡常地区水泥价格下调20-30元/吨,主要原因一是淡季临近,下游需求较弱,各企业发货量仅在6-7成,多数企业出现胀库现象;二是之前区域内价格多是小幅暗降,部分企业先行下调后与其他企业价差拉大;三是进入6月份,龙头企业为稳定客户,价格开始正式大幅回落。受安徽水泥进入价格下跌影响,南京市场到位价整体下调20-30元/吨。6月份为青奥会前最后一个月水泥需求期,若各企业竞相增加出货量,将导致价格继续下滑。江西地区稳中有落。</p><p></p><p>受高考、农忙和降雨影响,下游需求趋弱运行,南昌地区主导企业通过调整优惠政策,高标号水泥价格再次下探,幅度20元/吨。南昌地区价格基本到底部,后期若再大幅下跌,将会有企业亏损。安徽地区水泥价格弱势运行。</p><p></p><p>随着安徽沿江铜陵、芜湖和马鞍山等地水泥和熟料价格走低,5月末合肥地区水泥价格回落20-30元/吨。此外,近期合肥治理超载,企业运输成本上升,后期到位价将会出现高低不一,或将促使部分企业价格继续调整。6月初,安徽马鞍山、安庆、全椒和铜陵等沿江地区价格分别下调20-30元/吨。上海地区水泥价格平稳,下游需求平淡,由于江苏南部和安徽沿海价格下跌,进入上海水泥价格偏低,后期上海价格有继续下行风险。</p><p></p><p>中南地区P.O42.5水泥价格环比前一周下跌1.2%。端午节前后,珠三角地区雨水天气仍然较多,企业库存上升明显,英德基地库存至70%以上,为缓解压力,企业以优惠促销方式,出厂价格变相再次走低20-25元/吨。广西地区价格平稳为主,原计划6月5日价格上调20元/吨,但受降雨影响未能实现。</p><p></p><p>目前企业发货量在7-8成,库存普遍不高,熟料30%-40%,水泥50%-60%,后期需求恢复正常,企业或将继续推动价格上调。湖南地区价格稳中有落。长株潭地区水泥价格继续在底部徘徊。岳阳地区受外围低价水泥进入影响,水泥价格下调10-15元/吨。河南地区受农忙影响,企业库存环比增加。湖北武汉以及周边地区价格暂稳,鄂州地区低标号下调20-30元/吨。</p><p></p><p>西南地区P.O42.5水泥价格环比前一周上涨1.0%。贵阳和黔南地区水泥价格上调20元/吨,熟料上调40元/吨。进入6月份,随着雨水天气减少,下游需求环比有所增加,以及部分企业设备开始年中检修,在主导企业大力推动下,价格如之前预期开始走高。四川自贡和内江地区因需求下滑,部分企业水泥下调10元/吨,成都价格继续平稳运行,尽管市场进入需求淡季,但各主导企业均在做停产检修计划,并欲将价格适当上调,应对淡季价格走低。</p><p></p><p>西北地区价格稳中有落。甘肃平凉价格下调20-40元/吨,从市场反馈看,当地下游需求表现仍然较好,但由于之前海螺出故障设备恢复正常,供应增加,价格出现大幅回落。兰州区域受雨水天气影响,下游需求受挫,企业针对个别客户小幅下调10元/吨,其他价格暂稳为主。5月底,陕西宝鸡价格下调15-20元/吨,价格下调后,企业销量并未增加,目前市场成交依旧疲软;陕中地区水泥价格底部持稳,市场进入需求淡季,后期价格多是低位震荡为主。</p><p></p><p>水泥库容比环比上升0.2个百分点。</p><p></p><p>上周全国水泥库容比约为69.1%,较前一周上升0.2个百分点。其中华北、华东和中南地区库容比环比继续上升,幅度分别为0.4、0.4和1.0个百分点,东北和西南地区库容比环比分别下降1.9和0.6个百分点,西北地区保持平稳。</p><p></p><p>目前东北地区库容比仍然最高,约75.0%,西南地区最低,约为63.1%。</p><p></p><p>对上市公司的影响。</p><p></p><p>从基本面来看,由于水泥市场下游需求整体趋弱运行,全国水泥价格自四月中旬开始已经连续7周小幅下行,上周全国水泥库容比继续环比小幅上升至69.1%,较去年同期上升1.4个百分点。资金紧张问题短期难有明显好转,近期多地还将进入农忙期,水泥市场即将开始进入传统淡季,未来仍有继续下行的风险。长期来看,未来水泥窑协同处置废弃物有较大发展空间,有望促进水泥行业的转型升级。同时,若水泥行业化解过剩产能相关政策能够得到较好的落实,也将会促进水泥行业的结构调整和供需关系改善。个股方面长期关注具有低成本优势以及受益于未来行业资源整合的华东地区龙头企业<SPAN function copyToClipboard(txt) {  //复制网址
	if(window.clipboardData) {
		window.clipboardData.clearData();
		window.clipboardData.setData("Text", txt);
		alert("复制链接成功！");
	} else if(navigator.userAgent.indexOf("Opera") != -1) {
		window.location = txt;

		}
		var clip = Components.classes['@mozilla.org/widget/clipboard;1'].createInstance(Components.interfaces.nsIClipboard);
		if (!clip)
		return;
		var trans = Components.classes['@mozilla.org/widget/transferable;1'].createInstance(Components.interfaces.nsITransferable);
		if (!trans)
		return;
		trans.addDataFlavor('text/unicode');
		var str = new Object();
		var len = new Object();

		if (!clip)
		return false;
		clip.setData(trans,null,clipid.kGlobalClipboard);
		alert("复制链接成功！");
	}
