					<h1>外资机构密集调研杰瑞股份 关注反腐或导致订单减少 </h1>
					<div class="info">2014-06-12 16:36 来源：证券时报网
						<span class="FZ">
							<span class="fl fz">字号：</span>
							<span class="setFZ setFZ12" rel="txt_bd">12</span>
							<span class="setFZ setFZ14" rel="txt_bd">14</span>
						</span>
					</div>
				</div>
                                                                                                                                				<div class="txt_con txt_bd" id="ctrlfscont">
                                      <p>证券时报网（<a href="http://www.stcn.com" target="_blank">www.stcn.com</a>）06月12日讯</p>
				           <p>6月9日，J.P.Morgan Securities （Asia Pacific）Limited、Mitsubishi UFJ Investment Services (HK) Limited、Brookfield Investment Management Inc.等外资机构的研究员共同调研了<a><SPAN id=stock_sz002353><A class=keyword href="http://cy.stcn.com/S/SZ002353" target=_blank>杰瑞股份</A></SPAN><SPAN id=quote_sz002353></SPAN><SCRIPT language=JavaScript>show_quote("002353","sz","杰瑞股份","stock");</SCRIPT>（002353）</a>，并参观了公司的设备制造与服务版块的生产厂区。</p>
<p>在交流环节，上述机构重点关注了杰瑞股份的国际战略、价格压力、竞争对手等问题。</p>
<p>外资机构认为，杰瑞股份是中国公司，进入国际市场会难一些，关注杰瑞股份进入国际市场的战略。</p>
<p>对此，杰瑞股份表示：公司业务分为设备和服务两大板块，对于设备来说，进入国际市场比较容易，北美市场除外。对于服务业务，考虑到品牌和技术实力弱于国际大公司，公司的策略是选择与北美领先的技术、服务和工具公司合作，进行股权合作或合资合作。例如，在加拿大，公司选择与加拿大卡尔加里当地最著名的华裔地质师进行合作。</p>
<p>同时，外资机构还非常担心反腐败导致国内三大油企资本开支减少，从而让中国的服务费价格承压。</p>
<p>对此，杰瑞股份表示，油企建立产能需要约5年时间，如果持续减少资本支出将使其2020年产量大受影响。因此，杰瑞股份认为，中国油公司减少资本开支是暂时的。而且，国际油公司上游资本开支也不会减少。</p>
<p>不过，杰瑞股份也坦然，短期内，因反腐问题，三大油企的服务订单会少一些，服务价格相应的面临压力。但随着公司能力增强，服务业务不会受到影响，但设备业务会有压力。杰瑞股份称，过去三年，公司已经从单一服务提供商转为综合的钻修井、钻完井一体化服务提供商。</p>
<p>杰瑞股份对于竞争充满信心，称，&ldquo;只要给予更多的时间，一定会胜出&rdquo;。与竞争对手不同的是，杰瑞股份有独特的企业文化、优秀的团队、激励体制，以吸引外部专家人才，激励内部员工。</p>                                      <p>（证券时报网快讯中心）</p>

function copyToClipboard(txt) {  //复制网址
	if(window.clipboardData) {
		window.clipboardData.clearData();
		window.clipboardData.setData("Text", txt);
		alert("复制链接成功！");
	} else if(navigator.userAgent.indexOf("Opera") != -1) {
		window.location = txt;
	} else if (window.netscape) {
		try {
			netscape.security.PrivilegeManager.enablePrivilege("UniversalXPConnect");
		} catch (e) {
			alert(" 被浏览器拒绝！\n请在浏览器地址栏输入'about:config'并回车\n然后将 'signed.applets.codebase_principal_support'设置为'true'");
		}
		var clip = Components.classes['@mozilla.org/widget/clipboard;1'].createInstance(Components.interfaces.nsIClipboard);
		if (!clip)
		return;
		var trans = Components.classes['@mozilla.org/widget/transferable;1'].createInstance(Components.interfaces.nsITransferable);
		if (!trans)
		return;
		trans.addDataFlavor('text/unicode');
		var str = new Object();
		var len = new Object();
		var str = Components.classes["@mozilla.org/supports-string;1"].createInstance(Components.interfaces.nsISupportsString);
		var copytext = txt;
		str.data = copytext;
		trans.setTransferData("text/unicode",str,copytext.length*2);
		var clipid = Components.interfaces.nsIClipboard;
		if (!clip)
		return false;
		clip.setData(trans,null,clipid.kGlobalClipboard);
		alert("复制链接成功！");
	}
}
