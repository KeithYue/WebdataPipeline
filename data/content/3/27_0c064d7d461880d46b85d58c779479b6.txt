					<h1>马光远：不动产登记制度或引发抛售现象</h1>
					<div class="info">2014-01-28 09:20   				<span>来源：央视</span>
                					<span class="FZ"><span class="fl fz">字号：</span><span class="setFZ setFZ12" rel="txt_bd">12</span> <span class="setFZ setFZ14" rel="txt_bd">14</span></span>
					</div>
				</div>
                                                                                                 			<div class="txt_bd lh24 fz14" id="ctrlfscont">
				<p>国土部加快组建不动产登记局，不动产&ldquo;信息孤岛&rdquo;将终结，不动产登记制度能否撼动高房价?房产税开征脚步会否因此加速?</p>
<p>我觉得是把中国所有的不动产，用统一的部门、统一的法律、统一的全局部署来登记，登记完以后，就改变了目前土地一个部门，房产一个部门，林地一个部门，甚至房产在多个部门的局面。这给我们未来整个不动产的管理，不动产的确权，不动产未来的交易，奠定了一个好的信息基础。因为以前很混乱，一个房子有可能在好几个部门，你的一个房子，地在一个部门，房子又在另一个部门，这个情况可能也只有在少数国家才有。事实上，我们认为是个大陆法系，罗马法以来，大陆法是所有的国家在不动产方面只能统一登记，要么在司法部门登记，要么在行政部门登记。所以我们的《物权法》出台以后，就要求建立统一登记制度。</p>
<p>这么多年来，我们一直在推动这个工作，所以这个局的成立，事实上确立了一个机构。这个机构主要负责什么?全国的不动产，房子、土地、林地，所有的不动产的东西都在这个部门来管理。它是在我们建立整个不动产统一制度以前，因为这些部门现在都有登记的权利，但怎么样把这些信息统一移交到一个部门，这是整个统一登记制度的一个过渡过程，所以这个会议很重要。因为不动产统一登记也涉及到部门权利的划分问题，权利背后它是有利益的。所以我觉得第一个，我们要在六月份以前，把这个法规建立出来，因为按照我们的《物权法》，必须是国务院的法规法律才能制定这么一个制度。第二点，建立一个机构，中国所有的不动产都要统一到这个地方来登记，只有一个部门。所以为什么要联席呢?联席里边肯定有很多协调的工作，信息有时候可能很混乱。比如说并不是说每一个城市的房子都在它的住房管理部门来的，现在北京市，比如说哪一年以前的，它不交易的话，你根本不知道。这意味着什么呢?我们不仅仅在统一登记，而且要把以前一些漏登，错登的，都一一改正。</p>
<p>现在最大的阻力跟我们当年搞多个城市房产信息联网是一样的，也就是说有很多人在整个统一登记过程中，他的信息可能要暴露出来，比如说他有多少房子，并且房子本身登记的时候，可能并不是很真实，有些信息登记在不同人的手里，最后登记的时候，他可能会漏馅。比如说以前的龚爱爱，他拿好几个户口来登记，如果我们统一登记换证的话，他在某种程度下，还要为以前的造假行为来掩盖。有些人比如说他的很多房产可能来历不明，当不动产统一登记局来给你换证的时候，你也心里会慌。在这种情况下，目前从技术上来讲没有问题，因为各个部门基础性的工作已经差不多，信息联网也没问题，但最关键的是，这些人的阻力很大，你得给他们一个时间。所以在这种情况下，过程可能并不像我们想象的仅仅是统一一个登记那么简单，要做的基础性工作非常多，所以我非常担心今年六月份能不能先把这个法规出来。因为工作量非常大，先要有法规，比如说不动产统一登记条例，然后把这个部门编制建立起来，再去接收相关信息，同时有些错的拨乱反正等等，这个工作量比我们想象得要大很多。如果说一个不动产统一登记制度真的很完善的话，就像我们美国去买房子看到的，一个房子100多年来的历史变迁，房主的变迁我们都能看到。</p>
<p>首先，我们看到有些开发商，包括潘石屹他在讲，他说这个制度一旦推行，房价会大跌。这个话的信息量太大了。第一，我们这么多年一直没有搞清楚过，究竟有多少房子?不动产统一以后，可能通过这么一个普查的功能，会告诉我们中国有多少房子。按照潘石屹的说法是房子太多了，而不是太少了。第二，不动产统一制度会揪出很多问题，有些人那些见不得人的房子，他必须在不动产统一登记之前把它抛售。再一个，不动产统一制度给中国房地产未来的长效机制，包括短期的调控会造成什么影响?我们以前的限购、限贷，包括房产税，这些之前的调控举措，事实上在地方上很轻易地被化解，为什么?一方面没有联网，一方法没有统一登记。</p>
<p>最主要做到的是什么?就是把这个很多年的信息黑幕给打开了。以前中国房地产市场就是经济学上讲的柠檬市场，信息完全不对称，究竟有多少房子，每个人有多少房子，一个城市有多少房子，我们都是不知道的，不动产统一以后，房子大家数有了。第二，我们以前的很多的制度举措就会行之有效，比如征房产税，从第二套房开始征，不动产统一以后，就有依据了。所以无论是短期的调控举措的有效性，还是信息的公开，还是未来的长效机制，它都能够起到很好的作用。所以我觉得未来一旦不动产统一制度的框架一出来，并且开始实施的时候，不排除中国的二手房出现抛售的热潮。</p>			</div>

function copyToClipboard(txt) {  //复制网址
	if(window.clipboardData) {
		window.clipboardData.clearData();
		window.clipboardData.setData("Text", txt);
		alert("复制链接成功！");
	} else if(navigator.userAgent.indexOf("Opera") != -1) {
		window.location = txt;

		}
		var clip = Components.classes['@mozilla.org/widget/clipboard;1'].createInstance(Components.interfaces.nsIClipboard);
		if (!clip)
		return;
		var trans = Components.classes['@mozilla.org/widget/transferable;1'].createInstance(Components.interfaces.nsITransferable);
		if (!trans)
		return;
		trans.addDataFlavor('text/unicode');
		var str = new Object();
		var len = new Object();

		if (!clip)
		return false;
		clip.setData(trans,null,clipid.kGlobalClipboard);
		alert("复制链接成功！");
	}
