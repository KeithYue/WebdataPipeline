					<h1>国航付费选座争议：未来或成航空业常态</h1>
					<div class="info">2014-06-11 06:52   				<span>来源：21世纪经济报道</span>
                					<span class="FZ"><span class="fl fz">字号：</span><span class="setFZ setFZ12" rel="txt_bd">12</span> <span class="setFZ setFZ14" rel="txt_bd">14</span></span>
					</div>
				</div>
                                                                                                 			<div class="txt_bd lh24 fz14" id="ctrlfscont">
				&nbsp;&nbsp;&nbsp;&nbsp;本报记者孟岩峰北京报道</p><p>备受质疑的国航付费选座在短暂叫停后，再度悄然重启。</p><p>6月10日，21世纪经济报道记者向国航方面求证获悉，付费选座并没有停止，依然在推行中。“我们官网还挂着相关的条例与规定，并没有停止的说法。现在是试运行，未来怎么样还没有具体计划。”国航宣传部相关人士表示。</p><p>在国航官网上，已经恢复的公告显示可付费选择座位的航班旅行日期为2014年5月26日至2015年5月26日。适用航线为北京至休斯敦、旧金山、洛杉矶、法兰克福、巴黎及伦敦希思罗六条国际往返航线，旅客需要在购买国航客票后，距航班起飞48小时之前到指定销售渠道付费选择座位。</p><p>该公告一度引发争议不断，而在质疑声中，国航默默停止了付费选座业务，然而仅仅十天，国航又恢复该项业务。</p><p>目前，国内航空公司中，国航、海航及春秋航空都涉及到该项业务。</p><p>争议之后的重启</p><p>国航上述人士介绍，付费区域只是经济舱第四排至第十排座位的部分座位。这些区域包括靠窗或过道的位置，国际航线时间过长，此类座位更容易受到旅客青睐。选择此类座位则需要付费100元人民币或者等值的当地货币。白金卡及以上等级会员免费预定。</p><p>而海航类似产品则是“至优座选”，投放在包括北京至西雅图、芝加哥、波士敦、布鲁塞尔等8条国际航线上。选座区域为经济舱第一排及紧急通道中间的位置，费用为500元。</p><p>在国航付费选座受到争议暂停同时，海航此产品也默默暂停，客服给出的解释为系统升级。</p><p>民航专家綦琦在接受21世纪经济报道记者采访时表示，从企业行为来看，国航与海航的付费选择完全没有问题，这是市场化的行为，是服务细分而已。</p><p>而来自海航的说法也证明，此服务推出也受到市场欢迎，七成的可选座位被选。“我们目前只是试行阶段，因为时间短尚无太多的反馈。而且很多外航都有付费选择座位的服务，我们也只是先尝试。”国航人士表示。</p><p>民航资源网专家王疆民在其专栏文章中力挺这一市场化做法。“企业在市场中合法推出一个产品，都不是以自己的意志来决定的，而是根据供求关系和市场竞争的需求来决定的，是市场行为而已。”在其看来，作为航空运输，其核心服务就是将旅客从一地运往另一地，而其他服务都属于辅助性服务，旅客对一些服务有需求，一般通过付费就能得到。</p><p>春秋航空也是除国航与海航外有付费选择的航空公司。</p><p>春秋航空作为国内运营成功的廉价航空公司，其在每一条航线上都实施付费选座模式。实际上，廉价航空将各项服务进行菜单化服务，包括行李、餐食等等各项服务都采取收费模式。</p><p>春秋航空选择第一排是70元，第二排是50元，第三至第十一排则是40元，而如果通过网上选取则收费分别为45、35及25元。“我们鼓励网上选择座位，航空公司的票价应该是市场化的，只要不超过国家规定的最高票价，拆分成几个部分，可以说是符合市场需求的。”春秋航空新闻发言人张武安表示。</p><p>然而，北京尚公律师事务所律师、研究消费公益诉讼的法学博士左亚洛在接受媒体采访时则认为，根据我国价格法第二十三条规定：“制定关系群众切身利益的公用事业价格、公益性服务价格、自然垄断经营的商品价格等政府指导价、政府定价，应当建立听证会制度，由政府价格主管部门主持，征求消费者、经营者和有关方面的意见，论证其必要性、可行性。”其认为收费选择座位涉嫌实体和程序双重违法。</p><p>民航专家张起淮则认为国航作为拥有较大市场占有率的民航企业，与外航的性质不尽相同，所以此新政并不合理。</p><p>然而，争议之下，付费选择悄然重启或也说明了市场的选择。</p><p>收费或成常态</p><p>实际上，付费选择座位对于外航来说已是常态。</p><p>港龙及国泰航空自2009年起就已经有该项服务，美联航等公司也有类似服务。张武安表示，如果航空公司肯把票价降下来，跟廉航采取同样的菜单式收费，也是市场化的行为。</p><p>不过随着国航试水六条国际航线，綦琦认为收费选座将成为航空公司的趋势。</p><p>国航在6月10日还开通了北京至华盛顿的直航航线，成为继纽约、洛杉矶、旧金山、休斯敦、夏威夷及温哥华之后北美地区的第七个通航点。</p><p>而随着国际航线的增加，如果全部实施付费选座，无疑也是一部分收入。</p><p>近几年，国航快速布局国际航线。而中国公民的出境人数在2013年达到了9818万人次，占比18.04%。目前国航在国际航线加地区航线的收益已经占总收益的34.85%。</p><p>根据此前的公开报道，国航将在2014年达到北美航点8个，欧洲航点16个，亚洲航点15个，日、韩航点13个，大洋洲2个，地区航点4个。</p><p>国航规划发展部副总经理孙庾此前公开表示，国航的国际化最核心的是提升国际竞争力。第一是开拓国际市场，要有更多、更远、频率更高的国际航线网络。</p><p>而6月1日开始实施的公务机票改革，国航作为第一批公务机票采购平台，在国际机票上也是占尽先机。在公务机票改革中明确规定，除特殊原因并经单位内部核准的情形外，对于目的地国家（地区）有国内航空公司直达航班的，则要求选择国内航空公司航班出入境；无直达航班的，应选择国内航空公司到最邻近目的地国家（地区）进行中转。</p><p>而国航的国际航线也或将成为行业低迷下的业绩提升法宝。“现在也都是在试运行，是否会推广要看市场是否接受，如果市场接受，就会得到推广。”张武安表示。（编辑杨颢）			</div>

function copyToClipboard(txt) {  //复制网址
	if(window.clipboardData) {
		window.clipboardData.clearData();
		window.clipboardData.setData("Text", txt);
		alert("复制链接成功！");
	} else if(navigator.userAgent.indexOf("Opera") != -1) {
		window.location = txt;

		}
		var clip = Components.classes['@mozilla.org/widget/clipboard;1'].createInstance(Components.interfaces.nsIClipboard);
		if (!clip)
		return;
		var trans = Components.classes['@mozilla.org/widget/transferable;1'].createInstance(Components.interfaces.nsITransferable);
		if (!trans)
		return;
		trans.addDataFlavor('text/unicode');
		var str = new Object();
		var len = new Object();

		if (!clip)
		return false;
		clip.setData(trans,null,clipid.kGlobalClipboard);
		alert("复制链接成功！");
	}
