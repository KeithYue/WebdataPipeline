	                   <p style="TEXT-ALIGN: center; MARGIN: 0cm 0cm 0pt" class="MsoNormal" align="center">
  <span style="FONT-SIZE: 16pt; mso-bidi-font-size: 12.0pt" lang="EN-US">
    <a href="http://xianhuo159.blog.tianya.cn/">kdj指标的使用</a>
  </span>
</p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">（</span> 
<span lang="EN-US">KDJ</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">指标的中文名称是随机指数）</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span
style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　最早起源于期货市场。随机指标是由乔治·莱恩首创的，它在通过当日或最近几日最高价、最低价及收盘价等价格波动的波幅，反映价格趋势的强弱</span>

<span lang="EN-US">.</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　随机指标在图表上共有三根线，</span> 
<span lang="EN-US">K</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">线、</span> 
<span lang="EN-US">D</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">线和</span> 
<span lang="EN-US">J</span> 
<span
style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">线。随机指标在计算中考虑了计算周期内的最高价、最低价，兼顾了股价波动中的随机振幅，因而人们认为随机指标更真实地反映股价的波动，其提示作用更加明显。随机指标</span>

<span lang="EN-US">KD</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">线的意义：</span> 
<span lang="EN-US">KD</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">线称之为随机指标，</span> 
<span lang="EN-US">K</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">为快速指标，</span> 
<span lang="EN-US">D</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">为慢速指标，当</span> 
<span lang="EN-US">K</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">线向上突破</span> 
<span lang="EN-US">D</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">线时，表示为上升趋势，可以买进；当</span> 
<span lang="EN-US">K</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">线向下突破</span> 
<span lang="EN-US">D</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">线时，可以卖出，又当</span> 
<span lang="EN-US">KD</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">值升到</span> 
<span lang="EN-US">90</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">以上时表示偏高，跌到</span> 
<span lang="EN-US">20</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">以下时表示偏低。　太高就有下跌的可能，而太低就有上涨的机会。</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">一、</span> 
<span lang="EN-US">KDJ</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">的计算</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span lang="EN-US">(N</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">日收盘价</span> 
<span lang="EN-US">- N</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">日内最低价</span> 
<span lang="EN-US">)</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">÷</span> 
<span lang="EN-US">(N</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">日内最高价</span> 
<span lang="EN-US">- N</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">日内最低价</span> 
<span lang="EN-US">)</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">×</span> 
<span lang="EN-US">100 = N</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">日</span> 
<span lang="EN-US">RSV</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　当日</span> 
<span lang="EN-US">K</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">值</span> 
<span lang="EN-US">= 2/3</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">前</span> 
<span lang="EN-US">1</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">日</span> 
<span lang="EN-US">K</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">值</span> 
<span lang="EN-US">+ 
<span style="mso-spacerun: yes">&nbsp;</span> 1/3</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">当日</span> 
<span lang="EN-US">RSV 
<span style="mso-spacerun: yes">&nbsp;</span> ;</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　当日</span> 
<span lang="EN-US">D</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">值</span> 
<span lang="EN-US">= 2/3</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">前</span> 
<span lang="EN-US">1</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">日</span> 
<span lang="EN-US">D</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">值</span> 
<span lang="EN-US">+ 1/3</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">当日</span> 
<span lang="EN-US">K</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">值</span> 
<span lang="EN-US">;</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　当日</span> 
<span lang="EN-US">J</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">值</span> 
<span lang="EN-US">= 3</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">当日</span> 
<span lang="EN-US">D</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">值</span> 
<span lang="EN-US">- 2</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">当日</span> 
<span lang="EN-US">K</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">值</span> 
<span lang="EN-US">.</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　若无前一日</span> 
<span lang="EN-US">K</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">值与</span> 
<span lang="EN-US">D</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">值，则可分别用</span> 
<span lang="EN-US">50</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">来代替。</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　以</span> 
<span lang="EN-US">9</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">日为周期的</span> 
<span lang="EN-US">KD</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">线为例。首先须计算出最近</span> 
<span lang="EN-US">9</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">日的</span> 
<span lang="EN-US">RSV</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">值，即未成熟随机值，计算公式为</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　</span> 
<span lang="EN-US">9</span>

<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">以上为超买区，</span>

<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">

<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　</span> 
<span lang="EN-US">6</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">、</span>

<span lang="EN-US">100</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">或小于</span>

<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">指标为依据</span>

<span lang="EN-US">J</span>

<span lang="EN-US">100</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">或小于</span> 
<span lang="EN-US">10</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">时被视为采取买卖行动的时机。</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　</span> 
<span lang="EN-US">7</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">、</span> 
<span lang="EN-US">KDJ</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">本质上是一个随机性的波动指标，故计算式中的</span> 
<span lang="EN-US">N</span>

<span lang="EN-US">5</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">至</span> 
<span lang="EN-US">14</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">为宜，可以根据市场或商品的特点选用。不过，将</span> 
<span lang="EN-US">KDJ</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">应用于周线图或月线图上，也可以作为中长期预测的工具。</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　</span> 
<span lang="EN-US">KDJ</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">与布林线的综合运用</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
  <span
  style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　ＫＤＪ指标是超买超卖指标，而布林线则是支撑压力类指标。两者结合在一起的好处是：可以使ＫＤＪ指标的信号更为精准，同时，由于价格日Ｋ线指标体系中的布林线指标，往往反映的是价格的中期运行趋势，因此利用这两个指标来判定价格到底是短期波动，还是中期波动具有一定作用，尤其适用于判断价格到底是短期见顶（底），还是进入了中期上涨（下跌），具有比较好的效果。</span>
</p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
  <span
  style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　我们知道，布林线中的上轨有压力作用，中轨和下轨有支撑（压力）作用，因此当价格下跌到布林线中轨或者下档时，可以不理会ＫＤＪ指标所发出的信号而采取操作。当然，如果ＫＤＪ指标也走到了低位，那么应视作短期趋势与中期趋势相互验证的结果，而采取更为积极的操作策略。</span>
</p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
  <span
  style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　但要注意的是，当价格下跌到布林线下轨时，即使受到支撑而出现回稳，ＫＤＪ指标也同步上升，可趋势转向的信号已经发出，所以至多只能抢一次反弹。而当ＫＤＪ指标走上８０高位时，采取卖出行动就较为稳妥，因为当股价跌破布林线中轨后将引发布林线开口变窄，此时要修复指标至少需要进行较长时间的盘整，所以说无论从防范下跌风险，还是从考虑持有的机会成本来看，都不宜继续持有。</span>
</p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
  <span
  style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　最后，总结一下综合运用ＫＤＪ指标和布林线指标的原则：即以布林线为主，对价格走势进行中线判断，以ＫＤＪ指标为辅，对价格进行短期走势的判断，ＫＤＪ指标发出的买卖信号需要用布林线来验证配合，如果二者均发出同一指令，则买卖准确率较高。</span>
</p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">&nbsp;</p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
  <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">四、技术指标小结</span>
</p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
  <span
  style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　许多投资者往往会在指标运用过程中产生这样的疑惑：有时指标严重超买，价格却继续上涨；有时指标在超卖区钝化十几周而价格仍未止跌企稳。实际上，投资者在这里混淆了指标与价格的关系。指标不能决定市场的走向，价格本身才决定指标的运行状况。价格是因，指标是果，由因可推出果，由果来溯因则是本末倒置。事实上，最能有效体现市场行为的是形态，投资者首先应当从技术形态中分析市场参与者的心理变化并服从市场。在涨跌趋势未改变之前，不要试图运用指标的超买、超卖或钝化等来盲目断定市场该反弹、该回调了。我们应当灵活地运用技术指标，充分发挥其辅助参考作用</span>
</p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">

<span lang="EN-US">KDJ</span></p>

<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">

<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">月</span>

<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">六、</span>

<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">使用方法</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　日线ＫＤＪ是一个敏感指标</span> 
<span lang="EN-US">,</span>

<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">随机性强</span>

<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">经常发生虚假的</span></p>

<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　买、卖信号</span>

<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">使投资者根据其发出的买卖信号进行买卖时无所适从。运用周线ＫＤＪ与日线ＫＤＪ共同金叉选股法</span>

<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">就可以过滤掉虚假的买入信号</span> 
<span lang="EN-US">,</span> 
<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">找到高质量的成功买入信号。</span></p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
  <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　周线ＫＤＪ与日线ＫＤＪ共同金叉选股法的买点选择可有如下几种：</span>
</p>
<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
  <span style="FONT-FAMILY: 宋体; mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'">　　第一种买入法：打提前量买入法。</span>

<p style="MARGIN: 0cm 0cm 0pt" class="MsoNormal">
