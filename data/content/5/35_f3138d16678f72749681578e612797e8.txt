					<div class="artcl" id="contentDIV">
	                   <div>
  <p>
    <strong>
      <img style="MAX-WIDTH: 650px" alt="已不幸 但不可不争" src="http://i5.jrjimg.cn/201312/17/bbs_attach_img/bbs_attach_img_13872782054751.jpg" width="347" />
    </strong>
  </p>
  <p>
    <strong>
    <span style="FONT-FAMILY: Calibri">“
    <span style="COLOR: rgb(128,0,128)">哀其不幸，怒其不争</span>”这句话最早出自大文豪鲁迅之口</span>
    <span>
      <span style="FONT-FAMILY: Calibri">,</span>
    </span> 
    <span style="FONT-FAMILY: Calibri">是鲁迅在小说《孔乙己》中对孔乙己的描述，借此来表达作者对当时的国人一种无奈，同情和愤怒的感情。</span></strong>
  </p>
  <p>
    <strong>
    <span style="FONT-FAMILY: Calibri">哀是悲哀，惋惜，同情的意思，</span> 
    <span style="FONT-FAMILY: Calibri">“哀其不幸”</span> 
    <span style="FONT-FAMILY: Calibri">是说对某人的不幸遭遇感到悲哀。</span></strong>
  </p>
  <p>
    <strong>
    <span style="FONT-FAMILY: Calibri">怒是愤怒，遗憾的意思，</span> 
    <span
    style="FONT-FAMILY: Calibri">“怒其不争”是说对某人或者某种落魄境遇的不争气，不抗争而感到愤怒和遗憾。整句的意思是说：对某人的不幸遭遇感到悲哀，对他或者她不抗争，不争气而感到愤怒和遗憾。</span></strong>
  </p>
  <p>
    <strong>
      <span style="FONT-FAMILY: Calibri">今天我对博文对标题就说从此句话而来</span>
      <span>
        <span style="FONT-FAMILY: Calibri">, ‘</span>
      </span>
      <span style="COLOR: rgb(128,0,128)">
      <span style="FONT-FAMILY: Calibri">已不幸</span> 
      <span style="FONT-FAMILY: Calibri">但不可不争</span></span>
      <span>
        <span style="FONT-FAMILY: Calibri">”</span>
      </span>
      <span style="FONT-FAMILY: Calibri">意思就说作为中国对股民已经是很不幸</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">但是你要学会保护自己</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">自己也要争气</span>
      <span>
        <span style="FONT-FAMILY: Calibri">!</span>
      </span>
    </strong>
  </p>
  <p>
    <strong>
      <span style="FONT-FAMILY: Calibri">今天我在一个投资群中看见很多股民依然在执着对寻找着投资的机会</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">看见一些人依然乐观的看法</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">也难怪</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">只能说某某营造出来的陷阱依然很美</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">因为最近的资本市场的新政出台也是让人目不暇接</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">或者说是温水煮青蛙中让部分人丧失了警惕</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">不过这个市场依然还是有良知者发出了自己的声音</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">比如执着的尽管是屡败屡战的多头沙先生最新写的关于剖析打新和剖析优先股的文章很是详细和一针见血</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">希望写出此文的沙先生的争气从现在开始</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">既然已经看清问题的本质</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">那你为何还执着的看多或者建议广大投资看好中国的股市呢</span>
      <span>
        <span style="FONT-FAMILY: Calibri">?</span>
      </span>
    </strong>
  </p>
  <p>
    <span style="FONT-FAMILY: Calibri">
      <strong>沙先生写的</strong>
    </span>
    <span>
      <strong>
      <span style="FONT-FAMILY: Calibri">:</span> 
      <span style="COLOR: rgb(128,0,128)">
      <span style="FONT-FAMILY: Calibri">打新依然成利益集团盛宴</span> 
      <span style="FONT-FAMILY: Calibri">散户只能吃蛋糕屑</span></span>
      <span>
        <span style="FONT-FAMILY: Calibri">&gt;</span>
      </span> 
      <span style="FONT-FAMILY: Calibri">各位可以仔细看看</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">其最核心的观点就是</span>
      <span>
        <span style="FONT-FAMILY: Calibri">: “</span>
      </span>
      <span style="COLOR: rgb(128,0,128)">
        <span style="FONT-FAMILY: Calibri">按市值配售</span>
        <span>
          <span style="FONT-FAMILY: Calibri">=</span>
        </span>
        <span style="FONT-FAMILY: Calibri">超级大忽悠</span>
        <span>
          <span style="FONT-FAMILY: Calibri">,</span>
        </span>
        <span style="FONT-FAMILY: Calibri">优先股转股</span>
        <span>
          <span style="FONT-FAMILY: Calibri">=</span>
        </span>
        <span style="FONT-FAMILY: Calibri">新的大小非</span>
      </span>
      <span>
        <span style="FONT-FAMILY: Calibri">”</span>
      </span></strong>
    </span>
  </p>
  <p>
    <strong>
      <span style="FONT-FAMILY: Calibri">因此对这样对市场你只能选择抛弃</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">同时也不能寄希望于</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">除非某某制定政策对时候会改弦更张</span>
      <span>
        <span style="FONT-FAMILY: Calibri">,</span>
      </span>
      <span style="FONT-FAMILY: Calibri">但是可能吗</span>
      <span>
        <span style="FONT-FAMILY: Calibri">?</span>
      </span>
    </strong>
  </p>
  <p>
    <strong>
    <span style="FONT-FAMILY: Calibri">因此我今天在看见一些人还沉浸在虚幻之中时说道</span>
    <span>
      <span style="FONT-FAMILY: Calibri">:”</span>
    </span> 
    <span style="FONT-FAMILY: Calibri">我建议大家还是理智点比较好</span>
    <span>
      <span style="FONT-FAMILY: Calibri">,</span>
    </span>
    <span style="FONT-FAMILY: Calibri">要弄清楚市场下跌的本质</span>
    <span>
      <span style="FONT-FAMILY: Calibri">,</span>
    </span>
    <span style="FONT-FAMILY: Calibri">把我博文中说股票的那部分仔细看看</span>
    <span>
      <span style="FONT-FAMILY: Calibri">,</span>
    </span>
    <span style="FONT-FAMILY: Calibri">哪怕再多的人骂我也不碍事</span>
    <span>
      <span style="FONT-FAMILY: Calibri">,</span>
    </span>
    <span style="FONT-FAMILY: Calibri">但是我是凭良心说话的</span>
    <span>
      <span style="FONT-FAMILY: Calibri">.</span>
    </span>
    <span style="FONT-FAMILY: Calibri">现在到明年上半年股票真不能看多</span>
    <span>
      <span style="FONT-FAMILY: Calibri">!</span>
    </span>
    <span style="FONT-FAMILY: Calibri">即便市场在下跌中也会不时对给出个股对交易性机会</span>
    <span>
      <span style="FONT-FAMILY: Calibri">,</span>
    </span>
    <span style="FONT-FAMILY: Calibri">但是</span>
    <span>
      <span style="FONT-FAMILY: Calibri">90%</span>
    </span>
    <span style="FONT-FAMILY: Calibri">以上的股票是和大盘以及指数共振的</span>
    <span>
      <span style="FONT-FAMILY: Calibri">,</span>
    </span>
    <span style="FONT-FAMILY: Calibri">试想在座的各位能有几位有这个能力可以挖掘到只有百分之几的个股的机会</span>
    <span>
      <span style="FONT-FAMILY: Calibri">?”</span>
    </span></strong>
  </p>
  <p>
    <strong>
    <span style="FONT-FAMILY: Calibri">醒醒吧</span>
    <span>
      <span style="FONT-FAMILY: Calibri">,</span>
    </span>
    <span style="FONT-FAMILY: Calibri">各位</span>
    <span>
      <span style="FONT-FAMILY: Calibri">!</span>
    </span>
    <span style="FONT-FAMILY: Calibri">我愿呐喊</span>
    <span>
      <span style="FONT-FAMILY: Calibri">,</span>
    </span>
    <span style="FONT-FAMILY: Calibri">我愿疾呼</span>
    <span>
      <span style="FONT-FAMILY: Calibri">!</span>
    </span>
    <span style="FONT-FAMILY: Calibri">拿鲁迅先生所讲就是</span>
    <span>
      <span style="FONT-FAMILY: Calibri">:</span>
    </span> 
    <span style="FONT-FAMILY: Calibri">“
    <span
    style="COLOR: rgb(128,0,128)">假如一间铁屋子，是绝无窗户而万难破毁的，里面有许多熟睡的人们，不久都要闷死了，然而是从昏睡入死灭，并不感到就死的悲哀。现在你大嚷起来，惊起了较为清醒的几个人，使这不幸的少数者来受无可挽救的临终的苦楚，你倒以为对得起他们么？</span>”</span></strong>
  </p>
  <p>
    <strong>
      <span style="FONT-FAMILY: Calibri">“
      <span style="COLOR: rgb(128,0,128)">然而几个人既然起来，你不能说决没有毁坏这铁屋的希望</span>。”</span>
    </strong>
