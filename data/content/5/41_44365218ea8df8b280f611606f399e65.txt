                                          <div class="clear"></div>
                                          <h2><span class="span02">发布时间：<font class="font01">2014-02-28</font></span>
                                          <span class="span02">作者：<font class="font01">张程程</font></span>
                                          <span class="span02">来源：<font class="font01">能源经济网</font></span>
                                          <span class="news_info">字号：【<a id="big">大</a><a id="medium">中</a><a id="small" style="color: rgb(255, 0, 0);">小</a>】</span>

                                <span style="display:none"><p style=\"text-indent: 2em; text-align: center;\"><span style=\"color: rgb(0, 0, 0);\"><img src=\"http://cs.inengyuan.com/ueditor/php/upload/20140228/13935706391735.jpg\" title=\"光伏再探索.jpg\"/></span></p><p style=\"text-indent: 2em; text-align: left;\"><span style=\"color: rgb(0, 0, 0);\">（能源经济网记者张程程综合报道）2月27日消息，随着投资者进一步回到这个号称最不稳定的股票市场之一，光伏产业再次发亮。</span></p><p style=\"text-indent: 2em; text-align: left;\"><span style=\"color: rgb(0, 0, 0);\">正如《华尔街日报》的Chris Dieterich报道的那样，自2008年的一轮恶性破产后，光伏行业重新洗牌。同时，生产成本的下降，意味着这个行业的勇者们已预示到这个过去一直不稳定的股票或会大幅上涨。</span></p><p style=\"text-indent: 2em; text-align: left;\"><span style=\"color: rgb(0, 0, 0);\">包括在明尼苏达州和爱荷华州等地区在内的一些国家级政策，现在扮演着能够支撑投资者们信心的重要角色。</span></p><p style=\"text-indent: 2em; text-align: left;\"><span style=\"color: rgb(0, 0, 0);\">有个学派认为光伏的增长潜力才刚刚开始探索，并且可能会有如美国那样的更大型经济体出现。</span></p><p style=\"text-indent: 2em; text-align: left;\"><span style=\"color: rgb(0, 0, 0);\">预计2018年，全球风能和太阳能共同发电的份额将翻番至8%。在2040年，美国这两者的发电份额将增加一倍以上。</span></p><p style=\"text-indent: 2em; text-align: left;\"><span style=\"color: rgb(0, 0, 0);\">国际能源署说，高整合水平（起码两者年发电量能达到30%或以上）取决于一个国家的电力系统是否稳定，即在短期内不需要重大投资来满足它的需求。这需要大量的短期投资，以满足日益增长的电力需求，或着用来替换旧的资产。</span></p><p style=\"text-indent: 2em; text-align: left;\"><span style=\"color: rgb(0, 0, 0);\">国际能源署表示，因为他们可以从头开始构建一个灵活的系统，再加上考虑所赋予的投资权利，这意味着此种新兴经济体有着巨大的潜力。</span></p><p style=\"text-indent: 2em; text-align: left;\"><span style=\"color: rgb(0, 0, 0);\">而在欧洲——属于国际能源署的稳定力量，其面对的挑战和机会都是显而易见的。法国一家能源巨头法国燃气集团GDF，在欧洲光伏市场上名声大噪，其一举签下某内地高达204亿美元的光伏场地资产。</span></p><p style=\"text-indent: 2em; text-align: left;\"><span style=\"color: rgb(0, 0, 0);\">不过，该公司的首席执行官Gerard Mestrallet表示，该地区发电的前景仍然暗淡。根据《华尔街日报》的Géraldine Amiel的报告，他曾说，在欧洲可再生能源补贴将会把传统的电力生产转变成一个亏本生意。</span></p><p style=\"text-indent: 2em; text-align: left;\"><span style=\"color: rgb(0, 0, 0);\">Mestrallet先生的公司并非唯一一个对此表示很感兴趣的公司。在德国，向可再生能源转型是一个雄心勃勃且亟待开发的工程，德国最大的公用事业公司RWE和E.ON正在努力适应这种节奏。</span></p><p style=\"text-indent: 2em; text-align: left;\"><span style=\"color: rgb(0, 0, 0);\">不过，有专家表示，光伏产业是可以吸引投资，但为此付出代价的是传统供应商。</span></p></span>
