TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　另外，中国之声援引华尔街分析师的看法称，“在全球经济复苏、运油量不断增加的状况，油价已经在持续上升。如考虑到产油量目前严重有限的大背景，任何意外事件的发生都会造成全球石油恐慌，进而，油价暴涨的可能性会非常大。”</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><STRONG><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　<STRONG>欧美备受煎熬</STRONG></SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"></STRONG><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　中宇资讯分析师申涛多次对记者表示，如将伊朗核问题定义为“极端影响因素”，且爆发战争，预计国际油价至少推高至150美元/桶的历史峰值。这意味着，国际油价将重新回到2008年7月盘中创下的历史高点。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　据了解，美国的汽油价格去年曾达到4美元/加仑，这一度引起美国总统奥巴马的担忧。他曾直言，“现在美国的油价太高了。”有意思的是，对于处于竞选年的美国来说，汽油价格目前又成了共和党和民主党的舌战话题。奥巴马的竞争对手公开许诺将让美国的汽油价格下降，这立即遭到了奥巴马“嗤之以鼻”地回应。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　不过，目前不是只有美国深受高汽油价格的苦恼。申涛认为，中国的成品油零售上调窗口或最快将于3月8日前后正式打开，“届时，三地原油变化率或将已经走高至6%-8%”。</SPAN><BR style="WIDOWS: 2; -webkit-text-stroke-width: 0px"><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　对于欧洲来说，一些企业也备受油价煎熬。据外媒近期报道，“目前，瑞士大型炼油商PetroPlus集团丧失了所有信贷额度，法国LyonDell集团炼油厂由于油价飙涨即将关闭。”</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(0,0,0); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; <P><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(250,252,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/25px 宋体; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal;
