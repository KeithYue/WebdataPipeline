									<h1 class="entry-title">高油价再次来袭</h1>
										<div class="comments-link">
					<a href="http://blog.china5e.com/caoxi/archives/7689#respond" title="《高油价再次来袭》上的评论"><span class="leave-reply">发表回复</span></a>				</div><!-- .comments-link -->

				<div class="entry-content">
			<p><P><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">&nbsp;&nbsp;&nbsp; 时隔一年，高油价再次来袭，这恐怕是脆弱复苏中的美国经济最不想见到的情况之一。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　2012年，美国经济其实开局不错，一度居高不下的失业率1月份意外大降至8.3%，三年低点。“困难户”克莱斯勒公司，不但不再向政府伸手要钱，还宣布将设厂增产，制造业作为经济支柱似露曙光。新房开工量有所增加，房价悄然上涨，房地产业似乎做出姿态要迎接暖春。股市也上扬，QE3正在热议中……一切看上去很美。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　但是，不合时宜地，去年年初的场景再次上演：油价重返每桶百元上方。推动力还是在中东，只是去年是埃及和利比亚，今年是伊朗。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　伊朗作为石油输出国组织(欧佩克)第二大石油出口国，其地位和重要性自然与埃及和利比亚不可同日而语，其一举一动都牵动石油市场神经。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　霍尔木兹海峡军演，油价上涨，欧美采取经济制裁措施，油价上涨，以色列虎视眈眈，油价也上涨，伊朗国家石油公司停止向英法供油，油价还上涨。归根结底，国际原油市场无法承受伊朗石油供应缺位的假设。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　虽然对这个假设，美欧政客一直故作轻松。欧盟委员会负责能源事务的委员MarleneHolzner坚持嘴硬，说欧洲家底充盈，即使伊朗中断出口，也有足够的石油供应。但实际上，欧洲每天从伊朗进口51.5万桶石油，真想不出自7月1日启动对伊朗的石油禁令后，Holzner该如何弥补这个大缺口。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　相比较而言，美国底气要足，本来对中东石油的依赖度就低，自身又有丰富的油气田。但国际原油市场高度联动，纽约油价作为北美市场石油定价基准也在年初以来大幅上涨。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　目前，更具国际代表性的伦敦布伦特油价已经涨至每桶120美元以上，纽约油价则接近每桶105美元，都是9个月以来的最高水平。国际能源机构警告说，石油成本占全球经济总量的比例逼近2008年时的历史最高水平，危及经济复苏。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　高油价将从各个方面影响经济，美国经济的脆弱复苏随时可能毁于一旦。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　高油价一方面会转化为汽油价格上涨，挤压消费者支出。自年初以来，美国汽油零售均价已经涨了25美分，为每加仑3.57美元，接近历史同期最高水平。专家预计，汽油很快将冲破每加仑4美元，一旦突破4美元的重要心理价位，消费者的消费习惯就会改变，不再下馆子，不再买衣服，不再花钱度假。消费者支出减少之时，也就是经济复苏幻灭之日。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　另外，高油价会带来通胀压力，让美联储在制定政策时左右为难。如果推出QE3，那加剧通胀，如果不推出QE3，那经济持续疲软，就业增长难以为继。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　高油价还会挤压企业利润空间，让本来就面临信贷紧缩、需求疲弱的企业再次遭到高成本的重压。高油价还打击市场情绪，导致股市下跌。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　历史常常会惊人重演，去年年初时美国经济同样开局不错，但之后却在高油价和本国政治决策僵局的内外夹击下，增速逐季放缓。</SPAN><BR style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">　　当然，不排除国际能源机构像去年一样再次动用石油战略储备。但是，战略石油储备终究是杯水车薪，无法阻挡油价涨势。所以这次，摇摇晃晃的美国经济复苏只能寄希望于伊朗局势改善，因为所有人都清楚，一旦局势失控，油价必然飙升，这个后果将是不能承受之重。</SPAN></P><br />
<P><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 14px/24px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><SPAN style="WIDOWS: 2; TEXT-TRANSFORM. none; BACKGROUND-COLOR: rgb(255,255,255); TEXT-INDENT: 0px; DISPLAY: inline !important; FONT: 12px/20px Simsun; WHITE-SPACE: normal; ORPHANS: 2; FLOAT: none; LETTER-SPACING: normal; COLOR: rgb(34,34,34); WORD-SPACING: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">作者：乔继红　　来源：经济参考报</SPAN></SPAN></P></p>
